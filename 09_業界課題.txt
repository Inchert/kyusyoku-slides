■ 人手不足問題
================================================================================

● 現状
--------------------------------------------------------------------------------

【人手不足の状況】

給食業界の人手不足は深刻化の一途

+-------------------------------------------------------------+
|                                                             |
|   ・調理師の有効求人倍率: 約3倍（求職者1人に3件の求人）     |
|   ・パート・アルバイトの確保が困難                          |
|   ・特に早朝勤務の人材が不足                                |
|   ・地方での人材確保がさらに困難                            |
|                                                             |
+-------------------------------------------------------------+

● 人手不足の原因
--------------------------------------------------------------------------------

【人手不足の構造的要因】

+----------------------------------------+
| 1. 労働条件の厳しさ                    |
+----------------------------------------+
| ・早朝出勤（5-6時）                    |
| ・立ち仕事、体力的負担                 |
| ・土日祝の勤務（病院・施設）           |
| ・賃金水準が他業種比で低め             |
+----------------------------------------+
           |
           v
+----------------------------------------+
| 2. 労働人口の減少                      |
+----------------------------------------+
| ・少子高齢化による労働力人口減         |
| ・他業種との人材獲得競争               |
+----------------------------------------+
           |
           v
+----------------------------------------+
| 3. イメージの問題                      |
+----------------------------------------+
| ・「きつい」「汚い」のイメージ         |
| ・キャリアパスが見えにくい             |
+----------------------------------------+

● 事業分野別の人材確保難易度
--------------------------------------------------------------------------------
| 分野             | 難易度       | 理由                               |
|------------------|--------------|-----------------------------------|
| 病院給食         | 極めて高い   | 365日稼働、早朝・夜勤あり          |
| 高齢者施設給食   | 極めて高い   | 365日稼働、嚥下対応の専門性        |
| 学校給食         | 中程度       | 土日祝休み、夏休みあり             |
| 事業所給食       | 中程度       | 土日祝休みが多い                   |
| 保育園給食       | 中程度       | 土曜勤務あり                       |

--------------------------------------------------------------------------------

【企業が取り組む対応策】

+-------------------------------------------------------------+
| 【待遇改善】                                                |
+-------------------------------------------------------------+
| ・賃金アップ                                                |
| ・福利厚生の充実                                            |
| ・休日数の増加                                              |
+-------------------------------------------------------------+

+-------------------------------------------------------------+
| 【省人化・効率化】                                          |
+-------------------------------------------------------------+
| ・調理機器の自動化                                          |
| ・クックチル・クックフリーズ導入                            |
| ・完全調理品・半調理品の活用                                |
| ・配膳ロボットの導入                                        |
+-------------------------------------------------------------+

+-------------------------------------------------------------+
| 【採用の多様化】                                            |
+-------------------------------------------------------------+
| ・シニア人材の活用                                          |
| ・外国人労働者の採用                                        |

● コスト上昇の内訳
--------------------------------------------------------------------------------
| 項目       | 上昇率（2022-2024） | 要因                     |
|------------|---------------------|--------------------------|
| 食材費     | +15〜25%            | 原材料高騰、円安         |
| 人件費     | +5〜10%             | 最低賃金上昇、人手不足   |
| 光熱費     | +20〜40%            | エネルギー価格高騰       |
| 物流費     | +10〜15%            | 燃料費上昇、2024年問題   |

■ 価格転嫁の困難
================================================================================

● 給食業界特有の価格硬直性
--------------------------------------------------------------------------------

+-------------------------------------------------------------+
|            価格転嫁が困難な構造                             |
+-------------------------------------------------------------+
|                                                             |
|  【学校給食】                                               |
|   給食費 = 保護者負担                                       |
|   -> 値上げには保護者・自治体の理解が必要                   |
|   -> 政治的判断を伴うため時間がかかる                       |
|                                                             |
|  【病院給食】                                               |
|   入院時食事療養費 = 診療報酬で規定                         |
|   -> 国の制度で決まるため自由に価格変更できない             |
|   -> 2024年6月に30円引き上げ（約20年ぶり）                  |
|                                                             |
|  【介護施設給食】                                           |
|   食費 = 介護報酬・利用者負担                               |
|   -> 制度に縛られる部分が大きい                             |
|                                                             |
|  【事業所給食】                                             |
|   委託契約 = 長期契約が多い                                 |
|   -> 契約期間中の価格改定は困難なケースも                   |
|                                                             |
+-------------------------------------------------------------+

● コスト上昇 vs 価格転嫁のギャップ
--------------------------------------------------------------------------------

■ 2023年ホーユー問題
================================================================================

● 事件の概要
--------------------------------------------------------------------------------

+-------------------------------------------------------------+
|             2023年 ホーユー事業停止問題                      |
+-------------------------------------------------------------+
|                                                             |
|   【概要】                                                  |
|   2023年9月、広島県に本社を置く給食会社「ホーユー」が       |
|   突然の事業停止を発表                                      |
|                                                             |
|   【影響】                                                  |
|   ・約150の学校・病院・福祉施設に影響                       |
|   ・児童・生徒・患者への食事提供が一時ストップ             |
|   ・社会的に大きな問題として報道                           |
|                                                             |
|   【原因】                                                  |
|   ・食材費・人件費の高騰                                   |
|   ・価格転嫁ができなかった                                 |
|   ・資金繰りの悪化                                         |
|                                                             |
+-------------------------------------------------------------+

● 業界への影響
--------------------------------------------------------------------------------
| 影響           | 内容                               |
|----------------|-----------------------------------|
| 信頼性低下     | 給食会社への不安感増大             |
| 審査厳格化     | 自治体の入札審査で財務状況重視     |
| 業界再編       | 財務基盤の弱い企業への警戒感       |
| 価格見直し     | 適正価格への理解が進む             |

● 教訓
--------------------------------------------------------------------------------

【ホーユー問題の教訓】

+-------------------------------------------------------------+
| 1. 適正価格の重要性                                         |
|    ->「安ければ良い」ではなく持続可能な価格設定が必要       |
+-------------------------------------------------------------+
| 2. 財務健全性の確認                                         |
|    -> 発注者は受託企業の財務状況をチェックすべき            |
+-------------------------------------------------------------+
| 3. 緊急時の備え                                             |
|    -> 代替事業者の確保、BCP（事業継続計画）                 |
+-------------------------------------------------------------+
| 4. 業界全体での取り組み                                     |
|    -> 適正価格の啓発、コスト上昇への理解促進                |
+-------------------------------------------------------------+

■ ハーベストにとっての課題と対応

--------------------------------------------------------------------------------
