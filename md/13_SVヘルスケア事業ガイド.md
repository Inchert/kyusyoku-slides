## SVとは

```
┌─────────────────────────────────────────────────────────────┐
│          SV（スーパーバイザー）とは                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   英語で「管理者」「監督者」を意味する役職                  │
│                                                             │
│   給食業界では：                                            │
│   複数の事業所（病院・施設）を担当し、                      │

```

### 事業所責任者（チーフ）との違い

| 役割 | SV | 事業所責任者（チーフ） |
|------|-----|----------------------|
| **担当範囲** | 複数の事業所（5〜15ヵ所程度） | 1つの事業所 |
| **主な業務** | 統括管理、品質監査、人材育成 | 現場運営、調理管理、シフト管理 |
| **顧客対応** | 施設責任者・経営層との折衝 | 日常的な連絡・相談対応 |
| **勤務形態** | 巡回型（オフィス＋現場訪問） | 固定（担当事業所に常駐） |
| **報告先** | 支社長・エリア長 | SV |

## 業務詳細

### 1. 事業所巡回・品質管理

#### 巡回の頻度と目的

| 巡回種類 | 頻度 | 目的 |
|----------|------|------|
| **定期巡回** | 週1〜月2回/事業所 | 品質維持、スタッフ指導 |
| **抜き打ち巡回** | 月1回程度 | 実態把握、緊張感維持 |
| **問題発生時** | 随時 | トラブル対応、是正指導 |
| **衛生監査** | 月1回/事業所 | HACCP遵守確認 |

#### SVの収支管理業務

| 業務 | 内容 | 頻度 |
|------|------|------|
| **月次収支確認** | 事業所別P/Lの確認・分析 | 月1回 |

| **人件費管理** | シフト適正化、残業管理 | 週1回 |
| **予算進捗管理** | 予算vs実績の確認、是正 | 月1回 |
| **原価改善** | メニュー見直し、仕入れ改善 | 随時 |

## SVとして成功するために

### 心構え

```
┌─────────────────────────────────────────────────────────────┐
│              SVとして大切にすべきこと                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【現場第一主義】                                           │
│   ・デスクワークより現場を優先                              │
│   ・現場スタッフの声をよく聴く                              │
│   ・困っている現場を放置しない                              │
│                                                             │
│  【率先垂範】                                               │
│   ・自分が手本を見せる                                      │
│   ・できないことを指示しない                                │
│   ・言行一致                                                │
│                                                             │
│  【顧客視点】                                               │
│   ・施設（顧客）の立場で考える                              │
│   ・利用者（患者・入所者）の満足を最優先                    │
│   ・クレームを成長の機会と捉える                            │
│                                                             │
│  【スタッフへの配慮】                                       │
│   ・一人ひとりを尊重する                                    │
│   ・良いところを見つけて褒める                              │
│   ・困っているときは一緒に解決する                          │
│                                                             │
│  【自己成長】                                               │
│   ・常に学び続ける姿勢                                      │
│   ・フィードバックを受け入れる                              │
│   ・失敗から学ぶ                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 陥りがちな失敗と対策

| 失敗パターン | 対策 |
|-------------|------|
| **現場に行かなくなる** | 週の巡回計画を立て、必ず実行 |
| **数字だけを追う** | 現場の声を聴く時間を確保 |
| **上から目線になる** | 感謝・ねぎらいの言葉を忘れない |
| **トラブルを抱え込む** | 早めに上司・本部に相談 |
| **施設との関係悪化** | 定期面談を欠かさない |

### SVとしてのキャリアパス

```
【SVからのキャリアアップ】

SV（スーパーバイザー）
      │
      │ 実績を積む
      ↓
エリアマネージャー / 支社長
      │
      │ さらに昇進
      ↓
事業部長 / 本部スタッフ
      │
      ↓
経営層
```
