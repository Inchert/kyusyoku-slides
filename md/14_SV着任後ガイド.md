## SV着任の意味

### 現場研修からSVへ

```
┌─────────────────────────────────────────────────────────────┐
│              現場研修とSVの違い                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【現場研修中】                                              │
│   ・教わる立場                                               │
│   ・1つの事業所に所属                                        │
│   ・調理・配膳の作業者                                       │
│   ・失敗しても先輩がフォロー                                 │
│                                                             │
│        ↓ 着任 ↓                                             │
│                                                             │
│  【SV着任後】                                                │
│   ・教える・管理する立場                                     │
│   ・複数の事業所を担当                                       │
│   ・現場を「見る」立場                                       │
│   ・自分の判断が現場に影響                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### SV着任時の心構え

```
┌─────────────────────────────────────────────────────────────┐
│         SV着任時に持つべき3つの姿勢                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 1. 【謙虚さ】                                               │
│    ・現場経験はまだ浅い                                     │
│    ・ベテランスタッフから学ぶ姿勢を忘れない                 │
│    ・「SVだから」と偉ぶらない                               │
│                                                             │
│ 2. 【学び続ける姿勢】                                       │
│    ・わからないことは素直に聞く                             │
│    ・先輩SVの動きを観察する                                 │
│    ・失敗から学ぶ                                           │
│                                                             │
│ 3. 【責任感】                                               │
│    ・担当事業所は自分の責任                                 │
│    ・逃げない、言い訳しない                                 │
│    ・でも一人で抱え込まない                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 最初の30日間（着任〜1ヶ月目）

### 最初の1週間

```
【1週間目にやること】

┌─────────────────────────────────────────────────────────────┐
│ Day 1-2: 引継ぎ                                             │
├─────────────────────────────────────────────────────────────┤
│ □ 前任SVまたは上司から引継ぎを受ける                        │
│ □ 担当事業所リストを確認                                    │
│ □ 各事業所の基本情報を把握                                  │
│   ・施設名、住所、連絡先                                    │
│   ・食数規模                                                │
│   ・契約内容の概要                                          │
│   ・現場責任者（チーフ）の名前                              │
│   ・施設側担当者の名前                                      │
│ □ 過去のトラブル・懸案事項の確認                            │
│ □ 社用PC・スマホ・車両等の準備                              │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ Day 3-5: 事業所訪問開始                                     │
├─────────────────────────────────────────────────────────────┤
│ □ 全担当事業所を訪問（挨拶回り）                            │
│ □ 現場責任者（チーフ）と顔合わせ                            │
│ □ 施設側担当者へ着任挨拶                                    │
│ □ 厨房の状況を確認                                          │
│ □ スタッフ全員に挨拶                                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ Day 6-7: 情報整理                                           │
├─────────────────────────────────────────────────────────────┤
│ □ 訪問で得た情報を整理                                      │
│ □ 各事業所の課題・懸念点をリストアップ                      │
│ □ 優先的に対応すべき事項を特定                              │
│ □ 1週間の振り返り・上司への報告                             │
└─────────────────────────────────────────────────────────────┘
```

### 着任挨拶のポイント

```
【着任挨拶で伝えること】

■ 現場スタッフへ
┌─────────────────────────────────────────────────────────────┐
│ 「本日よりSVとして担当させていただく○○です。              │
│  ○ヶ月間の現場研修を経て、今回SVとして着任しました。       │
│  まだまだ勉強中ですので、                                   │
│  皆さんに教えていただくことも多いと思います。               │
│  一緒により良い給食を作っていきたいと思いますので、         │
│  よろしくお願いいたします。」                               │
│                                                             │
│ ポイント：                                                  │
│ ・謙虚に、でも堂々と                                        │
│ ・「教えてください」の姿勢                                  │
│ ・一緒に頑張りたい、というメッセージ                        │
└─────────────────────────────────────────────────────────────┘

■ 施設側担当者へ
┌─────────────────────────────────────────────────────────────┐
│ 「お忙しいところ失礼いたします。                            │
│  ハーベストの○○と申します。                                │
│  この度、御施設の担当SVとして着任いたしました。             │
│  前任の○○より引継ぎを受けております。                     │
│  何かございましたら、いつでもご連絡ください。               │
│  引き続きよろしくお願いいたします。」                       │
│                                                             │
│ ポイント：                                                  │
│ ・名刺を渡す                                                │
│ ・連絡先を伝える                                            │
│ ・「何かあればすぐ対応します」の姿勢                        │
└─────────────────────────────────────────────────────────────┘
```

## 31日〜60日目（2ヶ月目）

### 2ヶ月目の目標

```
┌─────────────────────────────────────────────────────────────┐
│              2ヶ月目のゴール                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 担当事業所の「通常運営」を把握する                       │
│     → 何が普通で、何が異常かがわかる                        │
│                                                             │
│  2. 小さな改善を1つ実行する                                  │
│     → 成功体験を積む                                        │
│                                                             │
│  3. 現場からの相談を受けられるようになる                     │
│     → 「この人に聞けばいい」と思ってもらう                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2ヶ月目にやること

```
【2ヶ月目の重点項目】

■ 課題への着手
┌─────────────────────────────────────────────────────────────┐
│ 1. 優先度の高い課題を1つ選ぶ                                │
│    ・影響が大きいもの                                       │
│    ・自分で解決できそうなもの                               │
│    ・短期間で成果が見えるもの                               │
│                                                             │
│ 2. 解決策を考える                                           │
│    ・現場の意見を聞く                                       │
│    ・先輩SVに相談する                                       │
│    ・上司に相談する                                         │
│                                                             │
│ 3. 実行する                                                 │
│    ・小さく始める                                           │
│    ・効果を確認する                                         │
│    ・うまくいったら展開する                                 │
└─────────────────────────────────────────────────────────────┘

■ 定期巡回の確立
┌─────────────────────────────────────────────────────────────┐
│ ・各事業所の訪問頻度を決める                                │
│   例）A事業所：週2回、B事業所：週1回                        │
│ ・訪問時のチェックポイントを決める                          │
│ ・訪問記録をつける習慣                                      │
└─────────────────────────────────────────────────────────────┘

■ 数字の把握
┌─────────────────────────────────────────────────────────────┐
│ □ 各事業所の月次食数を把握                                  │
│ □ 人件費・食材費の概算を把握                                │
│ □ 残業発生状況を把握                                        │
│ □ 欠員状況を把握                                            │
└─────────────────────────────────────────────────────────────┘
```

### 2ヶ月目に起きがちなこと

```
【2ヶ月目あるある と対処法】

■ 現場から「前のSVは〇〇だった」と言われる
┌─────────────────────────────────────────────────────────────┐
│ 対処：                                                      │
│ ・「そうだったんですね、教えてください」と受け止める        │
│ ・良い点は継続、改善すべき点は徐々に変える                  │
│ ・焦って自分のやり方を押し付けない                          │
└─────────────────────────────────────────────────────────────┘

■ 思ったように動いてくれないスタッフがいる
┌─────────────────────────────────────────────────────────────┐
│ 対処：                                                      │
│ ・まずは関係構築を優先                                      │
│ ・指示ではなくお願いベースで                                │
│ ・チーフに相談する                                          │
│ ・いきなり注意・指導はしない                                │
└─────────────────────────────────────────────────────────────┘

■ 施設から要望・クレームが来る
┌─────────────────────────────────────────────────────────────┐
│ 対処：                                                      │
│ ・まず話を聞く、謝る                                        │
│ ・一人で判断せず上司に相談                                  │
│ ・対応策を施設に報告                                        │
│ ・これはSVとしての経験になる                                │
└─────────────────────────────────────────────────────────────┘

■ 自分の判断に自信が持てない
┌─────────────────────────────────────────────────────────────┐
│ 対処：                                                      │
│ ・当然のこと、最初は誰でもそう                              │
│ ・迷ったら上司・先輩SVに相談                                │
│ ・小さな判断から経験を積む                                  │
│ ・失敗しても学べばOK                                        │
└─────────────────────────────────────────────────────────────┘
```

### 60日目のセルフチェック

```

       │       ・メールチェック
       │       ・当日の予定確認
       │       ・前日の報告確認
       │

       │       ・朝礼参加または状況確認
       │       ・チーフとの打合せ

       │       ・昼食時の状況確認
       │       ・施設担当者との面談
       │
12:30  ██████  昼食・移動
       │

       │       ・課題対応
       │       ・スタッフ面談
       │

## 困ったときの相談先

### 相談の使い分け

```
【誰に相談するか】

┌─────────────────────────────────────────────────────────────┐
│ ■ 現場の日常的な課題                                        │
│   → 現場責任者（チーフ）                                   │
│   → 先輩SV                                                 │
├─────────────────────────────────────────────────────────────┤
│ ■ 判断に迷う案件                                            │
│   → エリアマネージャー（上司）                             │
├─────────────────────────────────────────────────────────────┤
│ ■ 施設との関係・契約に関わる案件                            │
│   → エリアマネージャー                                     │
│   → 営業担当                                               │
├─────────────────────────────────────────────────────────────┤
│ ■ 衛生・品質に関わる重大案件                                │
│   → エリアマネージャー                                     │
│   → 品質管理部                                             │
├─────────────────────────────────────────────────────────────┤
│ ■ 人事・労務に関する案件                                    │
│   → エリアマネージャー                                     │
│   → 人事部                                                 │
├─────────────────────────────────────────────────────────────┤
│ ■ 自分のキャリア・悩み                                      │
│   → エリアマネージャー                                     │
│   → メンター（いれば）                                     │
└─────────────────────────────────────────────────────────────┘
```

### 相談の仕方

```
【上手な相談の仕方】

┌─────────────────────────────────────────────────────────────┐
│ 1. 状況を整理してから相談する                               │
│    ・何が起きているか                                       │
│    ・いつから、どの事業所で                                 │
│    ・影響範囲                                               │
├─────────────────────────────────────────────────────────────┤
│ 2. 自分の考えを持って相談する                               │
│    ・「どうしたらいいですか」だけでなく                     │
│    ・「こう考えていますが、いかがでしょうか」               │
├─────────────────────────────────────────────────────────────┤
│ 3. タイミングを選ぶ                                         │
│    ・緊急でなければ相手の都合を確認                         │
│    ・緊急なら遠慮なく連絡                                   │
├─────────────────────────────────────────────────────────────┤
│ 4. 結果を報告する                                           │
│    ・相談した件がどうなったか報告                           │
│    ・感謝の言葉を忘れない                                   │
└─────────────────────────────────────────────────────────────┘
```

## 90日後のキャリア

### 90日後の姿

```
┌─────────────────────────────────────────────────────────────┐
│             90日後に目指す姿                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  「一人前のSV」ではなく「成長中のSV」                        │
│                                                             │
│  ・基本的な業務は一人でこなせる                              │
│  ・困った時は適切に相談できる                                │
│  ・現場から信頼され始めている                                │
│  ・自分の課題が明確になっている                              │
│  ・次の成長に向けて動き出している                            │
│                                                             │
