```
【例文】

```
【例文】

## 想定質問と回答例

### 業界理解に関する質問

#### Q1: 給食業界の課題は何だと思いますか？

```
【回答のポイント】
・人手不足
・コスト上昇と価格転嫁の困難
・少子化による市場変化

【回答例】
最も大きな課題は人手不足だと考えます。早朝勤務など労働条件の厳しさから
人材確保が困難な状況にあります。また、食材費や人件費が上昇する一方で、
特に学校給食や病院給食では価格転嫁が難しい構造があります。
2023年のホーユー問題は、この課題の深刻さを示していると思います。
一方で、高齢者施設給食など成長分野もあり、効率化や省人化技術の
導入で対応していくことが重要だと考えます。
```

#### Q2: 給食業界を志望する理由は？

```
【回答例】
食は人の健康と幸せの基盤であり、給食はそれを社会的に支える
重要なインフラだと考えています。景気に左右されにくい安定性と、
高齢化社会において成長が見込める将来性にも魅力を感じています。
特に、子供たちの成長を支える学校給食や、高齢者の健康を支える
ヘルスケア給食は、社会貢献性の高い仕事だと思います。
```

### ハーベストに関する質問

#### Q3: ハーベストについて知っていることを教えてください

```
【回答のポイント】
・1960年設立、65年以上の歴史
・売上347億円、従業員約10,000名
・5つの事業（コントラクト、ヘルスケア、学校給食、保育園、ヨシケイ）
・ハーベストネクストという学校給食専業の子会社
・民設民営方式のパイオニア
・ISO9001・ISO22000認証
・横浜市中学校給食の受託

【回答例】
ハーベスト様は1960年設立、65年以上の歴史を持つ給食会社です。
売上高は約347億円で、業界上位15位圏内の中堅企業です。
特徴的なのは、学校給食の民設民営方式で実績を重ねていることで、
ハーベストネクスト株式会社として専業化されています。
また、コントラクト、ヘルスケア、保育園給食、ヨシケイと5つの事業を
展開する多角経営も強みだと理解しています。
最近では2026年度からの横浜市中学校給食で、日量最大3万食という
大型案件を受託されたとお聞きしています。
```

#### Q4: ハーベストの強みは何だと思いますか？

```
【回答例】
3つの強みがあると考えています。
1つ目は、学校給食の民設民営方式での豊富な実績です。
施設の設計・建設から運営まで一貫して対応できるのは大きな強みです。
2つ目は、多角経営による安定性です。学校給食だけでなく、
ヘルスケア、産業給食、ヨシケイ事業と幅広く展開することで、
リスク分散と安定経営を実現されています。
3つ目は、ISO9001とISO22000の認証取得による品質管理体制です。
入札での優位性だけでなく、お客様への安心感につながっています。
```

## 逆質問の例

### 面接で聞くべき質問

```
【入社後のイメージを確認する質問】
・入社後の配属先はどのように決まりますか？
・1日の業務の流れを教えてください
・新人教育・研修制度について教えてください

【キャリアに関する質問】
・キャリアパスについて教えてください
・将来的に複数の事業分野を経験することは可能ですか？
・管理職への登用はどのような基準ですか？

【会社の方針に関する質問】
・今後の事業展開の方向性を教えてください
・学校給食事業（ハーベストネクスト）の成長戦略は？
・ヘルスケア事業の強化について考えていることはありますか？

【職場環境に関する質問】
・職場の雰囲気を教えてください
・残業時間はどのくらいですか？
・休日の取りやすさはいかがですか？
```

### 避けるべき質問

```
【NG例】
✕ 給料はいくらですか？（初回面接では避ける）
✕ 有給は取れますか？（聞き方に注意）
✕ 残業はありますか？（ネガティブに聞こえる）
✕ 転勤はありますか？（消極的に聞こえる）

【OKな聞き方】
○ 福利厚生について詳しく教えてください
○ ワークライフバランスについて教えてください
○ 勤務地の希望は考慮されますか？
```
