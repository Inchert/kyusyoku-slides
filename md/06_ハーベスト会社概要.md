## 沿革

### 会社の歩み

```
【ハーベストの歴史】

1960年 ●─── 設立
        │
1970年代●─── 給食事業の基盤確立
        │
1980年代●─── 事業エリア拡大
        │
1990年代●─── ヨシケイ事業開始
        │     学校給食参入
        │
2000年代●─── ISO認証取得
        │     品質管理体制強化
        │
2010年代●─── ハーベストネクスト設立
        │     学校給食事業を分社化
        │
2020年代●─── 民設民営方式で実績拡大
        │     横浜市中学校給食受託
        ↓
       現在
```

| 年代 | 主な出来事 |
|------|-----------|
| **1960年** | ハーベスト株式会社設立 |
| **1970-80年代** | 給食事業の基盤確立、エリア拡大 |
| **1990年代** | ヨシケイ事業開始、学校給食市場参入 |
| **2000年代** | ISO9001・ISO22000認証取得 |
| **2010年代** | ハーベストネクスト株式会社設立（学校給食分社） |
| **2020年代** | 民設民営方式の実績拡大、横浜市中学校給食受託 |
