## 給食事業とは

給食事業は、企業・学校・病院・福祉施設などの**顧客施設内**において、食堂運営や食事提供を行うサービス事業。

正式には「**コントラクトフードサービス**」（契約給食）と呼ばれる。

```
【給食事業のイメージ】

┌─────────────────────────────────────┐
│         顧客施設（病院・学校など）     │
│  ┌─────────────────────────────┐  │
│  │       厨房・食堂スペース        │  │
│  │                               │  │
│  │   ← 給食会社が運営を受託 →   │  │
│  │                               │  │
│  │  調理スタッフ / 食材 / 運営   │  │
│  └─────────────────────────────┘  │
│                                     │
│     利用者（患者・生徒・社員など）    │
└─────────────────────────────────────┘
```

---

## 運営形態の分類

### 委託 vs 直営

| 形態 | 説明 | 割合 |
|------|------|------|
| **委託給食** | 外部の給食会社に運営を委託 | 増加傾向 |
| **直営給食** | 施設が自ら調理・提供 | 減少傾向 |

**委託化が進む理由**
- 人材確保の難しさ
- コスト削減ニーズ
- 専門ノウハウの活用
- 衛生管理の高度化

### 調理方式の分類

| 方式 | 説明 | メリット | デメリット |
|------|------|----------|-----------|
| **自校方式** | 各施設内で調理 | できたて提供、柔軟対応 | コスト高、人員必要 |
| **センター方式** | 一か所で大量調理→配送 | スケールメリット | 配送時間、温度管理 |
| **クックチル** | 調理後冷却→再加熱 | 人員効率、計画生産 | 設備投資必要 |

## 給食の6分野

給食市場は大きく**6つの分野**に分類される。

### 1. 事業所給食

```
対象: 企業・工場・官公庁の社員食堂
```

- **最大の市場規模**（約1.6兆円）
- 企業の福利厚生として位置づけ
- コロナ後の出社回帰で需要増

**主な形態**
- 社員食堂運営
- 工場内食堂
- オフィスカフェテリア

### 2. 病院給食

```
対象: 病院・診療所の入院患者
```

- 医療の一環としての「治療食」
- 管理栄養士による栄養管理が必須
- 診療報酬制度の影響を受ける

**特徴**
- 365日3食の提供が基本
- 個別対応（アレルギー、嚥下食など）
- 厳格な衛生管理

### 3. 高齢者施設給食

```
対象: 特養、老健、有料老人ホーム等
```

- **最も成長が期待される分野**
- 高齢化に伴い施設数増加
- 2028年には病院給食を上回る予測

**特徴**
- 嚥下機能に配慮した食形態
- 栄養ケアマネジメント
- 行事食・選択メニュー

### 4. 学校給食

```
対象: 小学校・中学校・高校
```

- 学校給食法に基づく実施
- 教育の一環としての「食育」
- 少子化により市場は横ばい

**運営形態**
- 自校方式（学校内で調理）
- センター方式（給食センターから配送）
- 民間委託が増加傾向

### 5. 幼稚園・保育所給食

```
対象: 保育園・幼稚園・こども園
```

- 待機児童対策で施設数増加も
- 少子化で長期的には減少傾向
- アレルギー対応の重要性

### 6. 弁当給食

```
対象: 在宅高齢者、オフィスワーカー等
```

- 配食サービス・宅配弁当
- 高齢者向け在宅配食が成長
- コロナ禍で注目度上昇

## 給食業界の特徴

### 労働集約型産業

```
売上の構成（イメージ）

┌─────────────────────────────────────┐
│█████████████████│ 食材費 約40%     │
├─────────────────────────────────────┤
│██████████████   │ 人件費 約35-40%  │
├─────────────────────────────────────┤
│█████           │ その他 約15-20%  │
├─────────────────────────────────────┤
│██              │ 利益 約5-10%     │
└─────────────────────────────────────┘
```

- **人件費比率が高い**（現場スタッフが多数必要）
- **薄利多売のビジネスモデル**
- 規模の経済が働きやすい

### 参入障壁

| 項目 | 難易度 | 理由 |
|------|--------|------|
| 資格要件 | 中 | 栄養士・調理師等の有資格者が必要 |
| 設備投資 | 中 | 厨房設備は施設側が用意することが多い |
| 実績・信用 | 高 | 入札・競争で実績が重視される |
| 人材確保 | 高 | 慢性的な人手不足 |

### 景気との関係

- **景気変動の影響を受けにくい**（必需サービス）
- ただしコスト（食材費・人件費）の変動影響は大きい
- 企業業績悪化時は社食縮小の可能性

## 業界の歴史

| 年代 | 出来事 |
|------|--------|
| 1950年代 | 学校給食法制定、給食の普及開始 |
| 1960年代 | 企業の社員食堂が広まる |
| 1970年代 | 給食委託会社の台頭 |
| 1980年代 | 病院給食の外部委託解禁 |
| 1990年代 | 学校給食の民間委託拡大 |
| 2000年代 | 高齢者施設給食の成長 |
| 2010年代 | 大手によるM&A・業界再編 |
| 2020年代 | コロナ禍→回復、人手不足深刻化 |
