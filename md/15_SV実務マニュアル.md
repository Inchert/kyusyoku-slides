| 食種 | 対象 | 提供頻度 |

| **術後食（5段階）** | 手術後の患者 | 非常に多い |
| **絶食→流動食** | 術前・術後・検査前 | 多い |
| **糖尿病食** | 糖尿病患者 | 多い |
| **腎臓病食** | 腎臓病患者 | 多い |
| **心臓病食（減塩）** | 心疾患患者 | 多い |
| **嚥下調整食** | 嚥下障害患者 | 中程度 |
| **アレルギー対応** | 食物アレルギー患者 | 中程度 |

### 老健でよくある対応

| 状況 | 対応例 |
|------|--------|
| **嚥下機能の改善** | 食形態のアップ（ペースト→やわらか→軟菜） |
| **嚥下機能の低下** | 食形態のダウン、とろみ追加 |
| **リハビリによる食欲増進** | 食事量の調整、間食の検討 |
| **在宅復帰準備** | 自宅での食事に近いメニュー提案 |
| **家族への指導** | 嚥下調整食の作り方、とろみの付け方 |

## 施設別 食事提供の違いまとめ

### 食事に求められるものの違い

```
【施設タイプ別 優先順位】

病院（急性期）
  1位: 治療効果（治療食の正確な提供）
  2位: 安全性（衛生管理・アレルギー）
  3位: 個別対応（食形態変更への迅速対応）

病院（療養型）
  1位: 栄養状態の維持
  2位: 食べやすさ（嚥下対応）
  3位: 飽きさせない工夫

老健
  1位: リハビリ支援（栄養管理）
  2位: 食形態の適切な評価・変更
  3位: 在宅復帰を見据えた支援

特養
  1位: 生活の質・楽しみ
  2位: 食べやすさ（嚥下対応）
  3位: 行事・季節感

有料老人ホーム
  1位: 満足度・おいしさ
  2位: 見た目・サービス
  3位: 選択肢の多さ
```

### 食種構成の違い

| 施設タイプ | 一般食 | 治療食 | 嚥下調整食 |
|------------|--------|--------|-----------|
| **急性期病院** | 30% | 50% | 20% |
| **療養型病院** | 20% | 30% | 50% |
| **老健** | 30% | 20% | 50% |
| **特養** | 20% | 10% | 70% |
| **有料老人ホーム** | 50% | 10% | 40% |

## 職種別の特徴と対応方法

### 事業所責任者（チーフ）への対応

```
┌─────────────────────────────────────────────────────────────┐
│           事業所責任者（チーフ）への対応                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【チーフの役割】                                           │
│   ・事業所全体の運営管理                                    │

│   ・良い点を認め、褒める                                    │
│                                                             │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### チーフとの面談で確認すべきこと

| 項目 | 確認内容 |
|------|----------|
| **人員状況** | 欠員はないか、退職予定者は、採用の必要性 |
| **労務状況** | 残業時間、有給取得、シフトの適正性 |
| **品質状況** | クレーム、ヒヤリハット、衛生面の課題 |
| **施設との関係** | 要望、クレーム、関係性 |
| **収支状況** | 食材費、人件費、原価の状況 |
| **悩み・課題** | チーフ自身が困っていること |

### トラブル発生時の対応優先順位

```
【同時多発トラブル時の優先順位】

優先度1（最優先で対応）

├── アレルギー事故
├── 施設からの緊急クレーム
└── 当日の欠員で提供不能の恐れ

優先度2（当日中に対応）
├── 設備故障（調理に支障）
├── 食材未納・発注ミス
├── スタッフ間のトラブル（深刻なもの）
└── 施設からの通常クレーム

優先度3（翌日以降でOK）
├── 採用面接の調整
├── 軽微なクレーム対応
├── 帳票の不備修正
└── 改善活動の推進

【判断に迷ったら】
・人の安全に関わること → 最優先
・施設との信頼に関わること → 高優先
・社内の問題 → 相対的に後回しOK
・迷ったら上司に電話で相談
```

## よくある失敗と対策

### 未経験SVがやりがちな失敗

| 失敗 | 原因 | 対策 |
|------|------|------|
| **上から目線になる** | 「管理する側」という意識 | 「サポートする側」と認識を変える |
| **現場に行かなくなる** | デスクワークに追われる | 巡回日を先にブロックする |
| **報告を後回しにする** | 忙しくて後回し | 当日中に必ず報告する習慣 |
| **分からないことを聞かない** | 聞きづらい | 「聞かないリスク」を理解する |
| **一人で抱え込む** | 迷惑をかけたくない | 早めに上司に相談する |
| **チーフを飛び越える** | 直接スタッフに指示 | 必ずチーフを通す |
