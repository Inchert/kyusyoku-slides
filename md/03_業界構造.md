## ビジネスモデルの基本

### 委託給食の仕組み

```
┌─────────────────────────────────────────────────────┐
│                    顧客施設                         │
│          （病院・学校・企業・福祉施設）              │
│                                                     │
│    ┌─────────┐      委託契約      ┌─────────┐     │
│    │ 施設側  │ ←───────────────→ │給食会社 │     │
│    │         │                    │         │     │
│    │・場所提供│                    │・人材派遣│     │
│    │・設備  │                    │・食材調達│     │
│    │・委託費 │                    │・調理運営│     │
│    └─────────┘                    └─────────┘     │
│         ↓                              ↓          │
│    ┌─────────────────────────────────────┐        │
│    │            厨房・食堂               │        │
│    │      給食会社のスタッフが運営       │        │
│    └─────────────────────────────────────┘        │
│                      ↓                            │
│              利用者へ食事提供                       │
└─────────────────────────────────────────────────────┘
```

### 直営 vs 委託の比較

| 項目 | 直営 | 委託 |
|------|------|------|
| 運営主体 | 施設自身 | 給食会社 |
| 人材確保 | 施設の責任 | 給食会社の責任 |
| コスト | 固定費が高い | 変動費化しやすい |
| 専門性 | 限定的 | 高い |
| 柔軟性 | 高い | 契約範囲内 |
| トレンド | 減少傾向 | **増加傾向** |

## 契約形態の種類

### 主な契約タイプ

#### 1. 管理費契約

```
【管理費契約の仕組み】

施設 → 給食会社
     │
     ├── 食材費（実費精算）
     │
     └── 管理費（固定）
         ├── 人件費
         ├── 事務費
         └── 利益
```

| 特徴 | 内容 |
|------|------|
| 食材費 | 実費精算（施設負担） |
| 管理費 | 月額固定 |
| リスク | 食材費変動は施設負担 |
| 適用 | 病院、福祉施設に多い |

#### 2. 食単価契約

```
【食単価契約の仕組み】

1食あたり ○○○円 × 提供食数 = 委託費
```

| 特徴 | 内容 |
|------|------|
| 単価 | 1食あたりの金額を設定 |
| 計算 | 単価 × 食数 |
| リスク | 食材費変動は給食会社負担 |
| 適用 | 学校給食、事業所に多い |

#### 3. 準直営型

```
【準直営型の仕組み】

施設の指示のもと、給食会社が人材を派遣
```

| 特徴 | 内容 |
|------|------|
| 形態 | 人材派遣に近い |
| 指揮命令 | 施設側 |
| 責任 | 施設側が大きい |

## 収益構造

### コスト構成

```
【売上に対するコスト構成（イメージ）】

100% ━━━━━━━━━━━━━━━━━━━━━━━━━━━
     ┃███████████████████│ 食材費      40%
 60% ┃──────────────────────────────────
     ┃██████████████     │ 人件費      35%
 25% ┃──────────────────────────────────
     ┃██████             │ その他経費   15%
 10% ┃──────────────────────────────────
     ┃███               │ 営業利益    5-10%
  0% ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 項目 | 構成比 | 内容 |
|------|--------|------|
| **食材費** | 約40% | 原材料費（原価率） |
| **人件費** | 約35-40% | 調理師、栄養士、パート等 |
| **その他経費** | 約15-20% | 水光熱費、消耗品、管理費 |
| **営業利益** | 約5-10% | 薄利構造 |

### 収益性の特徴

| 特徴 | 説明 |
|------|------|
| **薄利多売** | 利益率5-10%と低い |
| **規模の経済** | 大量仕入れでコスト削減 |
| **固定客** | 契約期間中は安定収入 |
| **参入障壁** | 実績・信用が重要 |

## バリューチェーン

### 給食会社の業務範囲

```
【バリューチェーン】

┌────┐   ┌────┐   ┌────┐   ┌────┐   ┌────┐
│メニュー│→│食材 │→│ 調理 │→│ 配膳 │→│ 清掃 │
│ 開発 │  │ 調達 │  │     │  │ 提供 │  │ 片付 │
└────┘   └────┘   └────┘   └────┘   └────┘
   │        │        │        │        │
   ↓        ↓        ↓        ↓        ↓
 栄養士   仕入先   調理師   配膳     洗浄
 管理    との交渉  パート   スタッフ  スタッフ
```

### 各工程の詳細

| 工程 | 担当 | 内容 |
|------|------|------|
| メニュー開発 | 管理栄養士 | 栄養計算、献立作成 |
| 食材調達 | 本部・エリア | 一括仕入れ、品質管理 |
| 調理 | 調理師・パート | 大量調理、衛生管理 |
| 配膳・提供 | 現場スタッフ | 温度管理、アレルギー対応 |
| 清掃・片付け | 現場スタッフ | 衛生管理、食器洗浄 |

## 顧客獲得の流れ

### 新規契約の獲得方法

#### 1. 入札・コンペ

```
【公共施設（学校・病院）の場合】

発注者（自治体等）
    │
    │ 入札公告
    ↓
┌─────────────────────────────┐
│  複数の給食会社が参加      │
│  ・価格                    │
│  ・実績                    │
│  ・提案内容                │
└─────────────────────────────┘
    │
    │ 評価・選定
    ↓
落札者決定 → 契約
```

#### 2. 紹介・営業

```
【民間施設（企業・福祉施設）の場合】

・既存顧客からの紹介
・営業活動
・コンサルタント経由
    │
    ↓
商談 → 提案 → 見積 → 契約
```

### 契約期間

| 施設タイプ | 一般的な契約期間 |
|------------|-----------------|
| 学校給食 | 1〜3年（入札） |
| 病院 | 1〜3年 |
| 企業食堂 | 1〜5年 |
| 福祉施設 | 1〜3年 |

## 競争環境

### 競争の軸

| 競争軸 | 内容 |
|--------|------|
| **価格** | コスト競争力、食単価 |
| **品質** | 味、栄養バランス、安全性 |
| **サービス** | 柔軟な対応、提案力 |
| **実績** | 類似施設での運営経験 |
| **人材** | 有資格者の数、定着率 |

### 差別化ポイント

- 特定分野への特化（病院専門、学校専門など）
- 付加価値サービス（栄養指導、イベント対応）
- IT活用（発注システム、栄養管理ソフト）
- 産地直送・地産地消
- アレルギー対応力
