### 詳細な食種分類

#### 一般食（段階別）

| 食種 | 対象 | 内容 |
|------|------|------|
| **常食** | 通常の咀嚼・嚥下機能 | 一般的な食事 |
| **軟菜食** | やや咀嚼困難 | やわらかめの食事 |
| **全粥食** | 米飯が困難 | 主食を粥に変更 |
| **五分粥食** | さらに消化に配慮 | 粥をやわらかく |
| **三分粥食** | 術後回復期など | 重湯に近い粥 |
| **流動食** | 経口摂取開始期 | 液体状の食事 |

#### 治療食（疾患別）

| 疾患 | 食種名 | 制限・調整内容 |
|------|--------|---------------|
| **糖尿病** | エネルギーコントロール食 | カロリー制限（1,200〜1,800kcal等） |
| **腎臓病** | たんぱく質制限食 | たんぱく質30〜50g/日、塩分制限 |
| **心臓病** | 塩分制限食 | 塩分6g/日以下 |
| **肝臓病** | たんぱく質調整食 | 状態に応じてたんぱく質調整 |
| **膵臓病** | 脂質制限食 | 脂質20〜30g/日 |
| **消化器疾患** | 低残渣食 | 食物繊維を控える |
| **術後** | 術後食 | 段階的に食形態をアップ |

#### 嚥下調整食（学会分類2021）

| コード | 名称 | 形態 | 対象 |
|--------|------|------|------|
| **コード0j** | 嚥下訓練食品0j | たんぱく質を含まないゼリー | 重度嚥下障害 |
| **コード0t** | 嚥下訓練食品0t | とろみ状 | 重度嚥下障害 |
| **コード1j** | 嚥下調整食1j | 均質でプリン・ゼリー状 | 嚥下障害 |
| **コード2-1** | 嚥下調整食2-1 | ピューレ・ペースト状 | 中等度嚥下障害 |
| **コード2-2** | 嚥下調整食2-2 | べたつかないまとまりやすい | 中等度嚥下障害 |
| **コード3** | 嚥下調整食3 | やわらか食 | 軽度嚥下障害 |
| **コード4** | 嚥下調整食4 | 軟菜食 | ごく軽度 |

## 運営方式の詳細

### 学校給食の4つの方式

```
【運営方式の比較】

┌────────────────────────────────────────────────────────────┐
│                    公設公営                                │
│  施設: 自治体所有  /  運営: 自治体職員                     │
│  → 減少傾向（コスト高）                                   │
├────────────────────────────────────────────────────────────┤
│                    公設民営                                │
│  施設: 自治体所有  /  運営: 民間委託                       │
│  → 最も一般的な委託形態                                   │
├────────────────────────────────────────────────────────────┤
│                    民設民営  ★ハーベストネクストの強み     │
│  施設: 民間所有    /  運営: 民間                          │
│  → 自治体の初期投資不要                                   │
├────────────────────────────────────────────────────────────┤
│                    PFI/DBO方式                             │
│  施設: PFI=民間建設 / DBO=公共建設                         │
│  運営: 民間                                                │
│  → 長期契約、包括委託                                     │
└────────────────────────────────────────────────────────────┘
```

### 各方式の詳細比較

| 項目 | 公設公営 | 公設民営 | **民設民営** | DBO | PFI |
|------|----------|----------|--------------|-----|-----|
| **施設所有** | 自治体 | 自治体 | **民間** | 自治体 | 民間→自治体 |
| **資金調達** | 自治体 | 自治体 | **民間** | 自治体 | 民間 |
| **設計・建設** | 自治体 | 自治体 | **民間** | 民間 | 民間 |
| **運営** | 自治体 | 民間 | **民間** | 民間 | 民間 |
| **契約期間** | - | 1-3年 | **10-20年** | 15-20年 | 15-20年 |
| **初期投資** | 大 | 大 | **不要** | 大 | 不要 |

```
              ↑ 年々上昇傾向

【少子化の影響】
児童生徒数減少 → 市場規模は横ばい
              → 1校あたりの競争激化
              → 民間委託の増加で相殺
```

### 今後の成長機会

| 機会 | 詳細 |
|------|------|
| **中学校給食の導入** | 選択制から全員喫食への移行（横浜市など） |
| **公設施設の老朽化** | 建て替え需要の増加 |
| **民設民営への移行** | 自治体の財政負担軽減ニーズ |
| **DBO方式の拡大** | 包括委託へのニーズ |
