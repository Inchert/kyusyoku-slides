## ハーベストを選ぶ理由

### 「なぜハーベストか」の回答例

```
【面接での回答例】

┌─────────────────────────────────────────────────────────────┐
│ 質問：「なぜ他社ではなくハーベストを選んだのですか？」      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 回答例1（ヨシケイ事業に注目）                               │
│ 「給食事業だけでなく、ヨシケイ事業という独自の強みを        │
│  持っている点に魅力を感じました。食を通じて施設だけでなく、 │
│  一般家庭にもサービスを届けている幅広さは、                 │
│  他社にはない特徴だと思います。」                           │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
```

## 業界共通の課題

### どの会社でも共通すること

```
【給食業界 共通の課題・特徴】

┌─────────────────────────────────────────────────────────────┐
│ ■ 業界共通の課題                                            │
├─────────────────────────────────────────────────────────────┤
│ ・人手不足（どの会社も採用に苦労）                          │
│ ・早朝・土日勤務がある（配属先による）                      │
│ ・立ち仕事が多い                                            │
│ ・給与水準は他業界と比べて高くない                          │
│ ・利益率が低い（薄利多売）                                  │
│ ・コスト上昇（食材費・人件費）の影響                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ ■ どの会社でも言えること                                    │
├─────────────────────────────────────────────────────────────┤
│ ・「配属先次第」で環境が大きく変わる                        │
│ ・口コミは参考程度（個人差が大きい）                        │
│ ・やりがいは「食で人を支える」こと                          │
│ ・資格（管理栄養士等）があると有利                          │
└─────────────────────────────────────────────────────────────┘
```
