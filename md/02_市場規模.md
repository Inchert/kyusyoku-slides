| 年度 | 市場規模 | 対2023年比 |
|------|----------|-----------|
| 2023年度 | 4兆7,915億円 | 基準 |
| 2026年度 | **5兆円超** | +5%程度 |
| 2028年度 | **5兆1,254億円** | **+7.0%** |

### 分野別の将来予測

#### 成長する分野

| 分野 | 2023→2028年 | 理由 |
|------|-------------|------|
| **高齢者施設給食** | 大幅増 | 高齢化、施設数増加 |
| 事業所給食 | 増加 | 出社回帰、福利厚生強化 |
| 弁当給食 | 増加 | 在宅高齢者向け配食 |

#### 横ばい・減少の分野

| 分野 | 2023→2028年 | 理由 |
|------|-------------|------|
| 学校給食 | 横ばい | 少子化 |
| 病院給食 | 減少 | 病床数削減、在院日数短縮 |
| 保育所給食 | 減少 | 少子化 |

## 注目ポイント：市場構成比の変化

### 2028年に「逆転」が起きる

```
【2023年度】              【2028年度】
┌────────────┐           ┌────────────┐
│  病院給食  │           │ 高齢者施設 │
│   (大)    │    →     │  給食(大)  │ ← 逆転！
├────────────┤           ├────────────┤
│ 高齢者施設 │           │  病院給食  │
│   給食    │           │           │
└────────────┘           └────────────┘
```

**背景**
- 日本の高齢化率：約30%（2025年）
- 有料老人ホーム・サ高住の増加
- 病院の機能分化・病床削減政策

## コロナ禍からの回復

### 市場への影響

```
【コロナ禍の影響と回復】

市場規模
    ┃
    ┃  ┌─┐
    ┃  │ │     ┌──────
    ┃  │ │  ┌──┘ 回復
    ┃  │ │  │
    ┃  │ │┌─┘ 底
    ┃  │ ││
    ┗━━━━━━━━━━━━━━━━━━
    2019 2020 2021 2022 2023 2024
         ↑
       コロナ
       ショック
```

| 時期 | 状況 |
|------|------|
| 2020年春 | コロナ禍で大幅落ち込み（特に事業所給食） |
| 2021年度〜 | 行動制限緩和で回復基調 |
| 2023年度 | **6分野すべてが前年比プラス** |
| 2024年度〜 | 引き続きプラス成長予測 |

### 分野別の影響度

| 分野 | コロナ影響 | 回復状況 |
|------|-----------|---------|
| 事業所給食 | 大（在宅勤務） | 出社回帰で急回復 |
| 学校給食 | 中（休校） | ほぼ回復 |
| 病院給食 | 小 | 安定 |
| 高齢者施設給食 | 小 | 安定成長 |
