● M&A活発化の背景

  +-------------------------------------------------------------------+
  |  【M&Aの目的】                                                     |
  +-------------------------------------------------------------------+
  |                                                                   |
  |  スケールメリット追求                                              |
  |  ・食材の大量仕入れ → コスト削減                                   |
  |  ・セントラルキッチン効率化                                        |
  |  ・人材採用力の強化                                                |
  |  ・営業エリアの拡大                                                |
  |  ・管理コストの削減                                                |
  |                                                                   |
  +-------------------------------------------------------------------+

【今後の予測】
  - 大手による中堅・中小の買収継続
  - 特定分野に強みを持つ企業が買収対象に
  - 業界上位への集約が進む
