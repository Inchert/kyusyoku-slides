■ トレンド2: 調理の効率化・自動化
================================================================================

● クックチル・クックフリーズの普及
--------------------------------------------------------------------------------

【調理方式の進化】

従来方式（クックサーブ）
+-------------------------------------------------------------+
|  調理 -> すぐに提供                                         |
|                                                             |
|  メリット: できたての味                                    |
|  デメリット: 人員が必要、ピーク時の負担大                  |
+-------------------------------------------------------------+
              |
              v 進化
クックチル方式
+-------------------------------------------------------------+
|  調理 -> 急速冷却（90分以内に0-3度）-> 冷蔵保管 -> 再加熱   |
|                                                             |
|  メリット: 計画生産可能、ピーク分散、人員効率化            |
|  デメリット: 設備投資、味の変化                             |
+-------------------------------------------------------------+
              |
              v さらに進化
クックフリーズ方式
+-------------------------------------------------------------+
|  調理 -> 急速冷凍（-18度以下）-> 冷凍保管 -> 再加熱         |
|                                                             |
|  メリット: 長期保存可能、計画生産、配送効率化              |
|  デメリット: 設備投資大、一部メニューに限定                |
+-------------------------------------------------------------+

● 省人化機器の導入
--------------------------------------------------------------------------------
| 機器                         | 用途             | 効果               |
|------------------------------|------------------|-------------------|
| スチームコンベクションオーブン | 焼く・蒸す・煮る | 調理の一括処理     |
| 真空包装機                   | 真空調理         | 品質安定、保存性向上|
| 自動炊飯システム             | 大量炊飯         | 省人化、品質均一   |
| 自動洗浄機                   | 食器洗浄         | 省人化、衛生向上   |
| 配膳ロボット                 | 料理の運搬       | 人員削減           |

================================================================================

● 市場拡大の背景
--------------------------------------------------------------------------------

【高齢者市場成長の構図】

高齢化率の上昇
    |
    +-- 高齢者施設の増加
    |   +-- 特養・老健
    |   +-- 有料老人ホーム
    |   +-- サ高住
    |
    +-- 在宅高齢者の増加
    |   +-- 配食サービス需要
    |
    +-- 嚥下調整食需要
        +-- 専門性の高い食事ニーズ

● 成長セグメント
--------------------------------------------------------------------------------
| セグメント         | 成長性   | ポイント                     |

| 高齢者施設給食     | 高成長   | 2028年に病院給食を逆転       |

| 冷凍宅配食         | 急成長   | 利便性、保存性               |
| 嚥下調整食         | 高成長   | 専門性ニーズ                 |
