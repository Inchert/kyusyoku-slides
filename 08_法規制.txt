■■ HACCP（ハサップ） ■■

●● HACCPとは ●●

+-------------------------------------------------------------+
|                    HACCP とは                               |
+-------------------------------------------------------------+
|                                                             |
|   Hazard Analysis and Critical Control Point                |
|   （危害要因分析重要管理点）                                |
|                                                             |
|   食品の安全性を確保するための衛生管理手法                  |
|                                                             |
|   【従来の衛生管理】      【HACCP】                         |
|    最終製品の抜き取り  →  製造工程全体を                   |
|    検査が中心             継続的に監視・記録                |
|                                                             |
+-------------------------------------------------------------+

●● HACCP制度化の経緯 ●●

┌──────────────┬────────────────────────────────────────────────┐
│ 年           │ 出来事                                           │
├──────────────┼────────────────────────────────────────────────┤
│ 2018年       │ 食品衛生法改正（HACCP制度化）                    │
│ 2020年6月    │ HACCP制度化施行                                  │
│ 2021年6月    │ 完全義務化（経過措置終了）                       │
└──────────────┴────────────────────────────────────────────────┘

現在の状況：
・原則すべての食品等事業者にHACCP対応が義務化
・給食事業者も対象

●● HACCPの7原則12手順 ●●

【HACCP導入の12手順】

+-------------------------------------------------------------+
| 【準備段階】手順1-5                                         |
+-------------------------------------------------------------+
| 手順1: HACCPチームの編成                                    |
| 手順2: 製品の記述                                           |
| 手順3: 意図する用途の確認                                   |
| 手順4: 製造工程一覧図（フローダイアグラム）の作成           |
| 手順5: 製造工程一覧図の現場確認                             |
+-------------------------------------------------------------+
                              │
                              ↓
+-------------------------------------------------------------+
| 【7原則】手順6-12                                           |
+-------------------------------------------------------------+
| 手順6 (原則1): 危害要因の分析                               |
| 手順7 (原則2): 重要管理点（CCP）の決定                      |
| 手順8 (原則3): 管理基準（CL）の設定                         |
| 手順9 (原則4): モニタリング方法の設定                       |
| 手順10(原則5): 改善措置の設定                               |
| 手順11(原則6): 検証方法の設定                               |
| 手順12(原則7): 記録と保存方法の設定                         |
+-------------------------------------------------------------+

●● 給食施設における重要管理点（CCP）の例 ●●

┌──────────────────┬──────────────────────┬────────────────────────────────┬────────────────────────┐
│ 工程             │ 危害要因             │ 管理基準                         │ モニタリング           │
├──────────────────┼──────────────────────┼────────────────────────────────┼────────────────────────┤
│ 加熱調理         │ 病原菌の残存         │ 中心温度75度以上1分以上          │ 温度計で測定・記録     │
│ 冷却             │ 細菌の増殖           │ 30分以内に中心温度20度以下       │ 温度・時間の記録       │
│ 保管             │ 細菌の増殖           │ 冷蔵10度以下、冷凍-18度以下      │ 庫内温度の記録         │
│ 配食             │ 細菌の増殖           │ 調理後2時間以内に喫食            │ 時間の管理             │
└──────────────────┴──────────────────────┴────────────────────────────────┴────────────────────────┘

●● 主な衛生管理基準 ●●

▼ 施設・設備

┌────────────────────────────┬─────────────────────────────────────────────────┐
│ 項目                       │ 基準                                              │
├────────────────────────────┼─────────────────────────────────────────────────┤
│ 調理場の区域分け           │ 汚染区域・非汚染区域の明確な区分                  │
│ 手洗い設備                 │ 適切な位置に十分な数を設置                        │
│ 温度管理                   │ 室温25度以下、湿度80%以下が望ましい               │
│ 換気                       │ 十分な換気能力の確保                              │
└────────────────────────────┴─────────────────────────────────────────────────┘

▼ 調理過程

┌────────────────┬─────────────────────────────────────────────────────────────────┐
│ 工程           │ 基準                                                            │
├────────────────┼─────────────────────────────────────────────────────────────────┤
│ 検収           │ 品温、鮮度、異物混入等を点検・記録                              │
│ 下処理         │ 十分な洗浄、流水で3回以上                                       │
│ 加熱調理       │ 中心温度75度以上で1分以上（二枚貝は85-90度90秒以上）            │
│ 配食           │ 調理後2時間以内に喫食                                           │
│ 検食           │ 原材料・調理済み食品を-20度以下で2週間以上保存                  │
└────────────────┴─────────────────────────────────────────────────────────────────┘

▼ 従事者の衛生管理

┌────────────────────────────┬─────────────────────────────────────────────────┐
│ 項目                       │ 基準                                              │
├────────────────────────────┼─────────────────────────────────────────────────┤
│ 健康診断                   │ 年1回以上の定期健康診断                           │
│ 検便                       │ 月2回以上（赤痢菌、サルモネラ、O157等）           │
│ 毎日の健康チェック         │ 発熱、下痢、嘔吐等の症状確認                      │
│ 手洗い                     │ 作業前、トイレ後、作業内容変更時等                │
│ 服装                       │ 専用の調理衣、帽子、履物                          │
└────────────────────────────┴─────────────────────────────────────────────────┘

■■ 大量調理施設衛生管理マニュアル ■■

●● 概要 ●●

+-------------------------------------------------------------+
|       大量調理施設衛生管理マニュアル                        |
+-------------------------------------------------------------+
|                                                             |
|   策定者: 厚生労働省                                        |
|   対象:   同一メニューを1回300食以上、または               |
|           1日750食以上を提供する調理施設                    |
|                                                             |
|   目的:   集団給食施設等における食中毒予防                  |
|                                                             |
+-------------------------------------------------------------+

●● 主な規定 ●●

┌──────────────────────────────┬─────────────────────────────────────────────────┐
│ 項目                         │ 内容                                              │
├──────────────────────────────┼─────────────────────────────────────────────────┤
│ 原材料の取り扱い             │ 検収記録、保管温度管理                            │
│ 加熱調理                     │ 中心温度75度1分以上（記録必須）                   │
│ 二次汚染防止                 │ 器具・容器の使い分け                              │
│ 温度管理                     │ 冷蔵10度以下、冷凍-15度以下                       │
│ 検食の保存                   │ -20度以下で2週間以上                              │

■■ 学校給食法 ■■

●● 概要 ●●

+-------------------------------------------------------------+
|                  学校給食法                                 |
+-------------------------------------------------------------+
|                                                             |
|   制定: 1954年（昭和29年）                                  |
|                                                             |
|   目的:                                                     |
|   ・学校給食の普及充実                                      |
|   ・学校における食育の推進                                  |
|                                                             |
|   【学校給食の目標】                                        |
|   1. 適切な栄養の摂取による健康の保持増進                   |
|   2. 日常生活における食事について正しい理解                 |
|   3. 学校生活を豊かにし、明るい社交性等の養成               |
|   4. 食生活が自然の恩恵の上に成り立つ理解                   |
|   5. 食生活が食に関わる人々の活動に支えられている理解       |
|   6. 伝統的な食文化についての理解                           |
|   7. 食料の生産、流通、消費についての理解                   |
|                                                             |
+-------------------------------------------------------------+

●● 栄養管理者の配置 ●●

┌────────────────────────────────┬─────────────────────────────────────────────────┐
│ 規模                           │ 配置基準                                          │
├────────────────────────────────┼─────────────────────────────────────────────────┤
│ 児童・生徒550人以上            │ 栄養教諭または学校栄養職員を配置                  │
│ 児童・生徒550人未満            │ 近隣校と兼務可能                                  │
│ 共同調理場                     │ 管理者1名以上配置                                 │
└────────────────────────────────┴─────────────────────────────────────────────────┘

■■ 食中毒予防 ■■

●● 主な食中毒菌と予防 ●●

┌────────────────────────────────┬────────────────────────────┬──────────────────────────────────────────────┐
│ 菌名                           │ 主な原因食品               │ 予防のポイント                                 │
├────────────────────────────────┼────────────────────────────┼──────────────────────────────────────────────┤
│ サルモネラ                     │ 卵、食肉                   │ 十分な加熱（75度1分以上）                      │
│ カンピロバクター               │ 鶏肉                       │ 十分な加熱、生肉からの二次汚染防止             │
│ 腸管出血性大腸菌（O157等）     │ 食肉、野菜                 │ 十分な加熱、手洗い、器具の消毒                 │
│ ノロウイルス                   │ 二枚貝、調理者からの汚染   │ 85-90度90秒以上加熱、手洗い徹底                │
│ 黄色ブドウ球菌                 │ 手からの汚染               │ 手指の傷がある場合は調理禁止                   │
│ ウェルシュ菌                   │ カレー、シチュー等         │ 加熱後の速やかな冷却、再加熱                   │
└────────────────────────────────┴────────────────────────────┴──────────────────────────────────────────────┘

●● 食中毒予防の三原則 ●●

+-------------------------------------------------------------+
|             食中毒予防の三原則                              |
+-------------------------------------------------------------+
|                                                             |
|   +------------------------------------------------------+  |
|   | 1. つけない（清潔）                                  |  |
|   |    ・手洗いの徹底                                    |  |
|   |    ・器具・容器の使い分け                            |  |
|   |    ・調理場の清潔保持                                |  |
|   +------------------------------------------------------+  |
|                                                             |
|   +------------------------------------------------------+  |
|   | 2. 増やさない（迅速・冷却）                          |  |
|   |    ・調理後速やかに喫食                              |  |
|   |    ・適切な温度管理                                  |  |
|   |    ・長時間の室温放置を避ける                        |  |
|   +------------------------------------------------------+  |
|                                                             |
|   +------------------------------------------------------+  |
|   | 3. やっつける（加熱）                                |  |
|   |    ・中心温度75度以上で1分以上加熱                   |  |
|   |    ・二枚貝は85-90度で90秒以上                       |  |
|   +------------------------------------------------------+  |
|                                                             |
