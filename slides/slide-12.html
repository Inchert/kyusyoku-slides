第12章 SVヘルスケア事業ガイド

SV（スーパーバイザー）とは

SV = 英語で「管理者」「監督者」を意味する役職

給食業界では：複数の事業所（病院・施設）を担当し、現場の運営管理・品質管理・人材育成を行う本部と現場の橋渡し役

別名：エリアマネージャー / エリア担当 / 巡回担当

ヘルスケア事業の特性

【対象施設】

  • 病院（急性期・回復期・慢性期・精神科）
  • 介護老人保健施設（老健）
  • 特別養護老人ホーム（特養）
  • 有料老人ホーム
  • サービス付き高齢者向け住宅（サ高住）
  • グループホーム
  • デイサービス

【特徴】

  • 365日3食の提供（休みなし）
  • 個別対応が多い（治療食・嚥下食・アレルギー）
  • 医療・介護スタッフとの連携が必須
  • 利用者の状態変化への迅速な対応
  • 厳格な衛生管理

SVの主要業務一覧

1. 事業所巡回・品質管理

  • 定期巡回（週1〜月2回/事業所）
  • 衛生管理チェック
  • 調理品質の確認
  • 記録・帳簿類の点検

2. 人材管理・育成

  • 事業所責任者の指導・育成
  • スタッフの教育・研修
  • 人員配置の調整
  • 採用活動のサポート

3. 顧客対応

  • 施設責任者との定期面談
  • クレーム対応・改善提案
  • 契約更新対応
  • 追加サービスの提案

4. 収支管理

  • 事業所別収支の管理
  • 食材費・人件費コントロール
  • 予算達成の進捗管理
  • 原価改善の提案・実行

5. 本部との連携

  • 本部方針の現場への展開
  • 現場課題の本部報告
  • 会議・報告書の作成
  • 各種データの集計・分析

6. トラブル対応

  • 緊急事態対応（食中毒リスク等）
  • 人員欠員時の対応
  • 顧客クレームの対応
  • 労務トラブルの対応

巡回時のチェック項目

【衛生管理】

  • 従業員の健康チェック記録
  • 検便実施状況
  • 手洗い・消毒の実施状況
  • 調理場の清掃状態
  • 冷蔵庫・冷凍庫の温度記録
  • 食材の保管状態（先入れ先出し）
  • 中心温度測定の記録
  • 検食の保存状況

【調理品質】

  • 献立通りの調理
  • 味付け・盛り付けの確認
  • 温度管理
  • 食形態の適切性（嚥下調整食）
  • アレルギー対応の確認
  • 治療食の正確な提供

【運営管理】

  • シフト表・勤務記録
  • 発注・在庫管理状況
  • 帳簿類の整備
  • ユニフォーム・身だしなみ
  • スタッフの態度・接遇
  • 施設側との連携状況

SVの評価指標（KPI）

収益関連

  • 食材費率 → 目標：40%以下
  • 人件費率 → 目標：35-40%
  • 営業利益率 → 目標：5-10%
  • 予算達成率 → 目標：100%以上

品質関連

  • 喫食率 → 目標：95%以上
  • 残食率 → 目標：5%以下
  • 衛生監査点数 → 目標：90点以上
  • クレーム件数 → 目標：月0件

顧客関連

  • 顧客満足度 → 目標：4.0以上（5点満点）
  • 契約継続率 → 目標：95%以上
  • 契約更新単価 → 目標：維持または増額

人材関連

  • 離職率 → 目標：年15%以下
  • シフト充足率 → 目標：100%
  • 有給取得率 → 目標：70%以上
  • 残業時間 → 目標：月20時間以下

SVに必要な5つのスキル

1. コミュニケーション能力

  • 施設責任者との折衝力
  • スタッフへの指導力
  • 本部への報告・提案力
  • 傾聴力、共感力

2. マネジメント能力

  • 複数事業所の統括
  • 人材育成
  • チームビルディング
  • 目標管理

3. 問題解決能力

  • 課題の発見・分析
  • 改善策の立案・実行
  • トラブル時の冷静な判断
  • 優先順位付け

4. 数値管理能力

  • 収支管理、原価管理
  • データ分析
  • Excel等の活用
  • 報告書作成

5. 専門知識

  • 給食運営の知識
  • 衛生管理（HACCP）
  • 栄養学の基礎
  • 労務管理の基礎

SVとして成功するために

大切にすべきこと

  • 現場第一主義：デスクワークより現場を優先
  • 率先垂範：自分が手本を見せる
  • 顧客視点：利用者の満足を最優先
  • スタッフへの配慮：一人ひとりを尊重する
  • 自己成長：常に学び続ける姿勢

陥りがちな失敗と対策

現場に行かなくなる週の巡回計画を立て必ず実行
数字だけを追う現場の声を聴く時間を確保
上から目線になる感謝・ねぎらいの言葉を忘れない
トラブルを抱え込む早めに上司・本部に相談

まとめ：SVの役割の本質

SV = 「現場の品質」と「会社の利益」を両立させる存在

  • ・現場が困らないようにサポートする
  • ・施設（顧客）に満足してもらう
  • ・会社として適正な利益を確保する
  • ・スタッフが成長できる環境を作る

このバランスを取ることがSVの腕の見せどころ

  • トラブルの芽を早期発見できているか

  • 自分自身も学び続けているか
