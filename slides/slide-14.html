第14章 SV実務マニュアル（未経験者向け）

未経験からSVへ：成功の5つのポイント

1
施設の違いを理解する
病院・老健・特養で求められるものが全く違う
2
現場スタッフを味方にする
上から目線ではなく一緒に問題解決
3
クライアントの立場で考える
施設が何を求めているかを常に意識
4
わからないことは正直に聞く
知ったかぶりは事故のもと
5
とにかく現場に足を運ぶ
現場を見ないと何もわからない

第1章：施設タイプ別の特徴

施設 入所者の状態 平均在所期間 食事の位置づけ SVとして注意すべき点
急性期病院 手術後・急性疾患 2週間程度 治療の一環 食事オーダー変更が頻繁、医療チームとの連携
療養型病院 長期療養患者 数ヶ月〜年単位 療養の支援 マンネリ防止、嚥下評価対応、栄養状態維持
老健 リハビリ中高齢者 3〜6ヶ月 リハビリ支援 食形態の頻繁な変更、在宅復帰支援、リハビリチーム連携
特養 要介護3以上 終身 生活の質 「食べる喜び」を支える、看取り期対応、行事食
有料老人ホーム 自立〜要介護 終身 生活の楽しみ 高いサービス要求、見た目・接遇徹底、選択メニュー

施設タイプ別 食事に求められるものの優先順位

急性期病院

  1. 治療効果（治療食の正確な提供）
  2. 安全性（衛生・アレルギー）
  3. 個別対応への迅速対応

療養型病院・老健

  1. 栄養状態の維持
  2. 食べやすさ（嚥下対応）
  3. 飽きさせない工夫

特養・有料老人ホーム

  1. 生活の質・満足度
  2. 見た目・サービス
  3. 行事・季節感

食種構成の目安

施設タイプ 一般食 治療食 嚥下調整食
急性期病院30%50%20%

第2章：従業員（現場スタッフ）への対応

事業所の人員構成（例：100食規模の施設）

事業所責任者（チーフ）：1名（正社員/管理栄養士or栄養士or調理師）

栄養士：1名 / 調理師：2〜3名（正社員・パート混在）

調理補助：3〜4名（パート） / 洗浄担当：2名（パート）

合計：10名程度（早番・遅番のシフト制）

職種別の対応方法

チーフへの対応

やるべきこと：

  • 信頼関係の構築が最重要
  • 定期的な面談（月1回以上）
  • 困っていることを引き出す
  • 権限委譲しつつサポート
  • 良い点を認め、褒める

やってはいけないこと：

  • チーフを飛び越えてスタッフに直接指示
  • チーフの面子を潰す言動
  • 細かいことまで口を出しすぎる

パート・調理補助への対応

パートさんの特徴：

  • 主婦層が中心（50-60代が多い）
  • 早朝勤務可能な方が貴重
  • 長く勤める方が多い

SVとしての関わり方：

  • 巡回時に必ず声をかける
  • 名前を覚えて呼ぶ
  • 「ありがとうございます」を伝える
  • ベテランパートさんを軽視しない

チーフとの面談で確認すべきこと

項目確認内容
人員状況欠員はないか、退職予定者は、採用の必要性
労務状況残業時間、有給取得、シフトの適正性
品質状況クレーム、ヒヤリハット、衛生面の課題
施設との関係要望、クレーム、関係性
収支状況食材費、人件費、原価の状況
悩み・課題チーフ自身が困っていること

よくある人間関係トラブルと対処法

パート同士の不仲

  1. チーフから状況ヒアリング
  2. 業務に支障があるか確認
  3. 必要に応じてシフト調整
  4. 深刻な場合は個別面談

ポイント：どちらかに肩入れしない

栄養士と調理師の対立

  1. 双方から個別にヒアリング
  2. 論点を整理（何が問題か）
  3. 必要に応じて三者面談
  4. 役割・責任の明確化

ポイント：専門性の違いを認め合う場を作る

ベテランが新人をいじめる

  1. 新人から状況ヒアリング
  2. ベテランの言い分も聴く
  3. チーフと対策を協議
  4. 必要に応じてベテランに指導

ポイント：新人を守る姿勢を示す、放置は絶対NG

第3章：クライアント（施設）への対応

施設側のキーパーソン

病院の場合

  • 事務長/管理部長：契約・お金の話
  • 栄養科長：日常連携、品質の話
  • 看護部長：患者さんからの声
  • 医師（NST担当等）：栄養管理の方針

介護施設の場合

  • 施設長/管理者：最終決定権者
  • 事務長：契約実務
  • 相談員/ケアマネ：入所者・家族の声
  • 介護主任：現場の声、食事介助状況

クレーム対応の基本

クレーム = 改善のチャンス

言ってもらえることに感謝 / 言わずに契約終了が最悪のパターン

味・品質に関するクレーム

  1. まず謝罪・傾聴
  2. 具体的な内容を確認（いつ、何が、どう）
  3. 現場確認・原因調査
  4. 改善策を提示（いつまでに、どう改善するか）
  5. フォローアップ訪問で状況報告

衛生面に関するクレーム

  1. 即座に現場へ急行
  2. 事実確認・証拠保全
  3. 施設責任者への報告・謝罪
  4. 本部への報告
  5. 原因究明・再発防止策
  6. 書面での報告・謝罪

※重大案件は最優先対応！

クレーム対応の言葉遣い

場面使うべき言葉避けるべき言葉
謝罪「申し訳ございません」「ご迷惑をおかけしました」「すみません」（軽い）
傾聴「おっしゃる通りです」「ご指摘ありがとうございます」「でも」「しかし」
説明「確認させていただきます」「調査いたします」「たぶん」「おそらく」
約束「○日までに改善いたします」「できるだけ早く」（曖昧）

第4章：複数店舗管理で成果を出す方法

SVの担当店舗数（目安）

新人SV（1年目）

3〜5事業所

中堅SV（2-3年目）

5〜8事業所

ベテランSV（4年目〜）

8〜12事業所

事業所のランク分けと巡回頻度

Aランク（要注意）

週2〜3回訪問

  • クレーム・トラブルが発生中
  • 新規オープン3ヶ月以内
  • チーフが新任・未熟
  • 施設との関係悪化中
  • 欠員が埋まらない

Bランク（通常管理）

週1回訪問

  • 安定しているが注視必要
  • チーフは育成中
  • 施設との関係は良好
  • 収支は概ね目標達成

Cランク（自走可能）

2週に1回訪問

  • チーフが優秀で自立
  • 施設との関係が非常に良好
  • 収支も安定して黒字
  • トラブルがほとんどない

営業・緊急対応

※Aランク事業所は週に複数回（月・木など）入れる / 金曜午後は緊急対応・営業用に空けておく

成果を出すSVの5原則

  1. 問題事業所に時間を集中させる - 均等配分はNG、Aランク事業所に週の40%を投下
  2. チーフを育てて任せる - 自分でやるより育てる、「どうしたらいい？」→「あなたはどう思う？」
  3. 予防に時間を使う - トラブル対応より未然防止、衛生チェック徹底で食中毒予防
  4. 数字を見る習慣を持つ - 収支・クレーム件数・欠員率・離職率を定期チェック
  5. 一人で抱え込まない - 困ったら上司・先輩に相談、新規オープン時は担当調整を依頼

SVが追うべき成果指標（KPI）

収益面

  • 担当エリアの営業利益
  • 食材原価率（目標：○%以内）
  • 人件費率（目標：○%以内）
  • 各事業所の収支

品質面

  • クレーム件数（目標：月○件以内）
  • 衛生監査の結果
  • 施設満足度調査の結果
  • 異物混入件数（目標：ゼロ）

人材面

  • 欠員率（目標：○%以内）
  • 離職率
  • 残業時間
  • 有給取得率

成長面

  • 契約継続率
  • 追加サービス獲得
  • 新規案件への貢献

緊急度×重要度マトリクス

第1領域（重要かつ緊急）

時間配分：20%

  • クレーム対応
  • 衛生トラブル
  • 欠員対応
  • 施設からの緊急要請

第2領域（重要だが緊急ではない）

時間配分：60%（最も重要！）

  • チーフ育成
  • 予防活動
  • 改善活動
  • 営業活動

陥りやすい失敗パターンと対策

失敗パターン

  • 巡回に追われて改善が進まない
  • 全事業所を均等に見てしまう
  • 自分で作業してしまう
  • 現場に行かなくなる
  • 数字を見ない

対策

  • 問題店舗に時間を集中
  • ランク分けでメリハリをつける
  • 作業ではなく管理・指導に徹する
  • 巡回日をブロックして死守
  • 週次で収支・KPIを確認する習慣

SV実務 日々のチェックリスト

  • 担当事業所の状況を把握しているか
  • ランク分けに基づいた巡回ができているか
  • チーフとのコミュニケーションは取れているか
  • 施設（顧客）との関係は良好か
  • 収支状況は把握しているか
  • トラブルの芽を早期発見できているか
  • 本部への報告・連絡は適切か
  • 予防活動に時間を使えているか
