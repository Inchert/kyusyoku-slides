Phase 1: 案件発掘

新規案件の種類

1. 入札案件

公立病院、自治体施設、学校給食センター

  • 公示→入札→落札という流れ
  • 価格だけでなく技術評価も重視（総合評価方式）
  • 実績・資格要件がある

2. 競合切り替え案件

現在他社が受託している施設

  • 施設のニーズ・不満を把握することが重要
  • 現受託者との比較になる
  • 契約更新時期を狙う

3. 新規開設案件

新しくオープンする施設

  • 早期の情報収集が重要
  • 設計段階から関われると有利
  • 開設準備から関わることで信頼構築

4. 直営→委託案件

現在直営で運営している施設

  • 委託のメリットを説明する必要
  • 既存職員の処遇が課題になることも
  • 初めての委託で不安を持っている

情報収集の方法

方法内容有効な案件
入札情報サイト官公庁の入札公告をチェック入札案件
業界紙・ニュース新規開設、移転情報新規開設案件
既存顧客からの紹介関連施設・知人の紹介全般
問い合わせ対応HPからの問い合わせ全般
飛び込み・テレアポ直接アプローチ競合切り替え
コンサルタント経由施設開設コンサルからの紹介新規開設案件

Phase 2: 提案・プレゼン

初回訪問（ヒアリング）の目的

  1. 施設のニーズを把握する
  2. 現状の課題を聴き出す
  3. 意思決定者・キーパーソンを把握する
  4. 競合状況を確認する
  5. スケジュール・予算感を確認する
  6. 信頼関係の第一歩を築く

ヒアリング項目

基本情報

  • 施設名・所在地
  • 施設種別（病院/老健/特養/有料等）
  • 定員・病床数
  • 開設時期（新規の場合）
  • 運営主体

給食に関する情報

  • 1日の提供食数（朝・昼・夕・おやつ）
  • 現在の運営形態（直営/委託/委託先）
  • 厨房設備の状況
  • 食種の構成
  • アレルギー対応の状況

ニーズ・課題

  • 委託を検討する理由
  • 現状の課題・不満
  • 特に重視するポイント
  • 過去のトラブル・懸念事項

意思決定・スケジュール

  • 意思決定者は誰か
  • 決定までのプロセス
  • 希望する開始時期
  • 予算感
  • 他社の検討状況

提案書の構成

1. 表紙・会社概要

  • 施設名、提案日、会社名
  • 会社紹介・理念
  • 類似施設の受託実績
  • 認証・資格（ISO、HACCP等）

2. 提案内容

  • 運営体制（人員配置）
  • 献立の考え方・サンプル
  • 衛生管理体制
  • 品質管理の仕組み
  • 緊急時対応

3. 施設の課題への対応

  • ヒアリングで把握した課題への具体策
  • 差別化ポイント

4. 見積金額

  • 月額費用
  • 内訳（食材費、人件費、管理費等）
  • 前提条件

5. スケジュール

  • 契約から運営開始までの流れ
  • 準備期間に必要なこと

プレゼンのポイント

準備

  • 出席者を事前に確認
  • 時間配分を決める（説明20分、質疑20分等）
  • 想定質問への回答を準備
  • 配布資料を人数分用意
  • 試食を持参する場合は準備

当日

  • 早めに到着
  • 笑顔で、はきはきと
  • 専門用語を使いすぎない
  • 相手の反応を見ながら進める
  • 質問には誠実に回答（分からないことは持ち帰る）
  • 最後に次のステップを確認

施設タイプ別 営業のポイント

施設 キーパーソン 重視されること 提案のコツ
病院 事務長、栄養科長 治療食対応力、衛生管理、実績 NST参画、栄養管理への貢献をアピール
老健 施設長、事務長 リハビリ支援、嚥下対応、価格 栄養ケアマネジメントへの貢献
特養 施設長、理事長 食の楽しみ、嚥下対応、行事食 「入所者の笑顔」を生む食事をアピール
有料老人ホーム 本部、施設長 サービス品質、見た目、選択メニュー 入居者満足度向上、施設の差別化に貢献

Phase 3: 契約締結

条件交渉のポイント

価格交渉

  • 安易な値下げはNG（収益性を確保）
  • 値下げ要請には理由を確認
  • 代替案を提示（サービス内容調整、段階的導入等）
  • 適正価格の根拠を説明（食材費、人件費の内訳）
  • 本部と相談しながら進める

その他の交渉事項

  • 契約期間（1年/3年/5年等）
  • 更新条件・解約条件
  • 価格改定条項（食材費高騰時の対応）
  • 設備・備品の負担区分
  • 光熱費の負担区分

契約書のチェックポイント

項目確認すべきこと
業務範囲何を受託するか明確か
委託費金額、支払条件、消費税
契約期間開始日、終了日、更新条件
解約条件解約予告期間、違約金
価格改定改定条件、協議条項
損害賠償範囲、上限

Phase 4: オープン準備

開業までのスケジュール

3ヶ月前

  • 人員採用計画の策定
  • 責任者の選定・配置決定
  • 献立作成開始
  • 設備・備品リストの作成
  • 施設との打ち合わせ（週1回）

1ヶ月前

  • スタッフ研修
  • 設備・備品の搬入
  • 試作・試食会
  • 施設スタッフへの説明会
  • 発注システムの準備

2週間前

  • リハーサル（実際の調理・提供）
  • 最終確認
  • 問題点の洗い出し・対応

オープン後

  • オープン初日
  • SVの常駐（最初の1週間）
  • 課題の早期発見・対応

オープン準備チェックリスト

人員

  • 責任者の配置完了
  • 栄養士の配置完了
  • 調理師の配置完了
  • パート・アルバイトの採用完了
  • シフト表の作成完了
  • 全員の研修完了

設備・備品

  • 調理機器の動作確認
  • 食器・トレイの準備
  • 調理器具の準備
  • 洗剤・消耗品の準備
  • ユニフォームの準備

運営体制

  • 献立の決定
  • 食材仕入先との契約
  • 発注システムの稼働
  • 衛生管理マニュアルの整備
  • 帳票類の準備
  • 緊急連絡体制の構築

施設との調整

  • 入所者情報の受領（食種、アレルギー等）
  • 配膳時間の確認
  • 連絡体制の確認
  • 試食会の実施
  • 施設スタッフへの説明完了

Phase 5: 運営開始・フォロー

立ち上げ期のSVの動き

初日〜1週間

  • SVが現場に常駐（または毎日訪問）
  • 調理・提供の流れを確認
  • 施設担当者への報告（毎日）
  • 課題の早期発見・即時対応
  • スタッフのフォロー

2週目〜1ヶ月

  • 週2-3回の訪問
  • 施設担当者との定期面談
  • 改善点の洗い出し・対応
  • スタッフの習熟度確認
  • 本部への報告

1ヶ月〜3ヶ月

  • 週1回の訪問
  • 月次振り返り（施設との面談）
  • 収支状況の確認
  • オペレーションの安定化
  • 通常管理体制への移行

ハーベストの差別化ポイント（プレゼン時にアピール）

  • 65年以上の歴史と実績
  • ISO9001・ISO22000認証
  • 多様な事業展開による安定性（ヘルスケア、学校給食、ヨシケイ等）
  • 「もっと食べたいと思える」食事へのこだわり
  • 嚥下調整食への対応力
  • 類似施設での具体的な実績

新規営業 成功のポイント

  1. 情報収集を怠らない → 入札情報、新規開設情報を常にチェック
  2. ニーズをしっかり聴く → 施設が何を求めているかを把握
  3. 自社の強みを明確に伝える → ハーベストならではの価値を提案
  4. 適正な価格で勝負する → 安売りは長続きしない
  5. オープン後のフォローを徹底する → 最初の印象が信頼を決める
  6. 既存顧客を大切にする → 紹介につながる、競合参入を防ぐ
