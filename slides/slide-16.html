第16章 緊急時対応マニュアル

第1部：緊急時対応の基本原則

SVとしての責務

緊急事態が発生した時、SVは現場とクライアント、そして本部をつなぐキーパーソンとして機能します。パニックにならず、冷静に状況を把握し、適切な判断と行動を取ることが求められます。

1. 情報収集・状況把握

迅速に情報を集め、何が起きているか正確に把握

  • 事実と推測を分ける
  • 5W1Hを意識
  • 時系列で整理

2. 初動対応の指示

現場への適切な指示、二次被害の防止

  • 優先順位を明確に
  • 具体的に指示
  • 現場へ急行

3. 報告・連携

上位への報告、施設・関係機関との連携

  • エスカレーション
  • 記録の確保
  • 再発防止策立案

SVが果たすべき7つの役割

  1. 迅速な情報収集と状況把握
  2. 適切な初動対応の指示
  3. 上位への報告（エスカレーション）
  4. 施設・クライアントへの連絡
  1. 二次被害の防止
  2. 記録の確保
  3. 再発防止策の立案

急な欠員 中 応援手配、業務調整、シフト組み替え 状況に応じて

第2部：食中毒発生時の対応

食中毒は最も深刻な事態

食中毒が発生した場合、入所者・患者の健康被害に加え、営業停止処分、損害賠償、マスコミ報道など、会社全体に甚大な影響を及ぼす可能性があります。迅速かつ適切な初動対応が極めて重要です。

食中毒対応フローチャート

STEP1：初動対応（発生から30分以内）

現場チーフがやること

  • □ 該当食品の提供を即時停止
  • □ 残存食品・検食の確保・保全
  • □ 発症者の状況確認
  • □ 施設側への第一報
  • □ SVへの連絡

絶対にやってはいけないこと

  • ✕ 検食・残存食品を廃棄する
  • ✕ 調理記録・温度記録を廃棄する
  • ✕ 原因を推測で発言する
  • ✕ 報告を遅らせる
  • ✕ 隠蔽しようとする

STEP2：情報収集（発生から2時間以内）

確認・収集すべき情報

  • □ 発症者数・症状の確認
  • □ 発症時刻・喫食内容の特定
  • □ 共通メニューの洗い出し
  • □ 調理記録・温度記録の確保
  • □ 調理担当者の健康状態確認
  • □ 原材料の仕入れ元・ロット確認

SVの具体的アクション

第一報を受けたら（〜30分）

  1. 現場へ急行する（可能な限り）
  2. 現場責任者から状況を聞き取る
    • 発症者数、症状、発症時刻
    • 喫食メニュー、提供時刻
  3. エリアマネージャーへ報告
  4. 施設側責任者と面談

現場到着後（〜2時間）

  1. 検食・残存食品の確保状況を確認
  2. 調理記録、温度記録、検収記録を確保
  3. 調理工程の聞き取り
  4. 調理スタッフの健康状態確認
  5. 施設側と今後の対応を協議
  6. 代替食の手配（必要に応じて）

保健所対応

保健所の立入検査に備える

  • 保健所の立入検査に必ず立ち会う
  • 記録類を求められたら提示する
  • 検便検査の手配をする
  • 質問には正確に回答（憶測で答えない）
  • 営業停止命令が出た場合の対応を検討

重要：わからないことは「確認します」と答え、決して推測で回答しない

検食の重要性

検食とは

提供した食事を一定期間保存しておくもの。食中毒発生時の原因究明に不可欠です。

項目 基準
採取量 各メニュー50g以上
保存温度 -20℃以下で冷凍保存
保存期間 2週間以上
ラベリング 日付、食事区分、メニュー名を記載

第3部：異物混入発生時の対応

異物の分類

危険異物（即時対応・報告必須）

健康被害のリスクが高い異物

  • 金属片（刃物の欠け、ホチキス針等）
  • ガラス片
  • プラスチック片（硬質）
  • 骨片（魚骨・肉骨）
  • 害虫・害虫の一部

→ 本部へ即時報告必須

非危険異物（報告・原因究明必要）

不快感を与えるが健康被害リスクは低い

  • 毛髪
  • 繊維（ふきん、ユニフォーム等）
  • ビニール片（軟質）
  • 野菜くず（皮、ヘタ等）
  • 紙片

→ 当日中に報告・原因究明

異物混入報告書の記載項目

記載項目内容
1. 基本情報発生日時、発生場所（施設名、病棟等）、発見者情報
2. 異物の詳細異物の種類・大きさ、混入していた料理名、写真添付
3. 発生状況発見の経緯、喫食の有無、健康被害の有無
4. 原因分析推定される混入経路、調理工程の確認結果
5. 対応内容発見者への対応、施設への報告内容
6. 再発防止策即時対応、恒久対策、実施予定日

第4部：機器故障時の対応

主要機器の故障対応

スチームコンベクションオーブン故障

影響：焼き物・蒸し物の大量調理が不可

代替：

  • 回転釜での蒸し調理
  • オーブンレンジの活用
  • メニュー変更（煮物等へ）

冷蔵庫・冷凍庫故障

影響：食材の品質劣化・廃棄リスク

代替：

  • 他の冷蔵庫への移動
  • 近隣事業所への食材避難
  • クーラーボックス＋保冷剤

判断：温度上昇した食材は使用禁止

食器洗浄機故障

代替：手洗い洗浄（80℃以上ですすぎ）、使い捨て食器の使用

ガス設備・水道トラブル

代替：電気調理機器活用、非加熱メニュー、外部からの調達

機器故障時のSVの判断ポイント

  1. 食事提供への影響度を判断 - 次の食事に間に合うか？代替調理方法はあるか？
  2. 代替手段の検討 - 他機器での代替、近隣事業所からの応援、外部からの調達
  3. 施設への連絡・相談 - 状況説明、代替案の提示、施設の意向確認
  4. 修理手配 - メンテナンス会社への連絡、修理見積・期間の確認

第5部：災害時の対応

地震発生時の対応

発生直後（〜5分）

  1. 身の安全確保（火を消す、テーブル下へ）
  2. 揺れが収まったら
    • スタッフの安否確認
    • 火元確認・ガス元栓閉める

初動対応（5分〜30分）

  1. 被害状況確認（人的被害、設備破損、ライフライン）
  2. 施設側との連携（避難指示に従う）
  3. 本部への報告（安否、被害、食事提供可否）

停電時の対応

即時対応

  • 冷蔵庫の扉を開けない（数時間は温度維持可能）
  • 調理中の料理の状況確認
  • 施設と復旧見込みを確認

停電時間別対応

  • 短時間（〜2時間）：ガス機器で調理継続
  • 長時間（2時間以上）：温度上昇食材は廃棄、備蓄食品活用

感染症発生時（施設内）

施設で感染症が発生した場合

  • 施設から情報収集（感染者数、感染エリア）
  • 厨房内の感染防止強化（手洗い徹底、体調不良者出勤停止）
  • 配膳方法の変更（使い捨て食器、エリア制限）

厨房スタッフに感染疑いが出た場合

  • 即時出勤停止
  • 濃厚接触者の特定
  • 厨房の消毒
  • 施設・本部への報告
  • 人員補充の手配

第6部：人員トラブル対応

急な欠勤への対応

代替要員の確保（優先順位）

優先度対応策特徴
1同事業所の他スタッフに出勤依頼現場を知っている、即戦力
2近隣事業所からの応援会社のルールを知っている
3派遣・スポットワーカー手配時間がかかる場合あり
4SV自身が現場に入る最後の手段、他業務に影響

スタッフ間トラブル

SVとしての対応

  1. 事実確認 - 双方から個別に話を聞く、第三者からも情報収集、憶測で判断しない
  2. 対応検討 - 軽微：注意・指導 / 深刻：シフト調整 / ハラスメント：本部人事への報告
  3. フォローアップ - 定期的な状況確認、改善が見られない場合は上位へエスカレーション

よくあるクレームと対応例

クレーム対応ポイント
「味が薄い/濃い」ご意見を受け止める。治療食の場合は医師の指示であることを説明。可能な範囲で調整検討。
「冷めている」即座に謝罪。温め直しまたは作り直しを提案。温蔵設備の温度設定を見直し。
「量が少ない」盛り付け量を確認。治療食の場合は栄養管理上の必要性を説明。施設栄養士と相談。
「同じメニューばかり」献立サイクルを説明。バリエーション改善を検討。選択食の導入を相談。

第8部：まとめ

緊急時対応の心得

緊急時のSV 5つの心得

1. 冷静に状況を把握する

パニックにならない、事実を確認

2. 報告は迅速に、正確に

悪い情報ほど早く上げる

3. 記録を残す

後で必ず必要になる

4. 一人で抱え込まない

上司・本部に相談

5. 再発防止を必ず行う

同じことを繰り返さない

実践編：緊急時の対応セリフ集

食中毒疑い発生時のセリフ

【現場から第一報を受けた時】

落ち着いて確認：「わかりました。落ち着いて状況を教えてください」

詳細確認：「何人が、いつから、どんな症状ですか？」

指示：「該当の食事の提供を止めてください。検食は絶対に捨てないでください」

対応表明：「今からそちらに向かいます。○分で着きます」

【施設責任者への謝罪】

第一声：「このたびは大変申し訳ございません」

状況報告：「現在把握している状況をご報告いたします」

対応方針：「保健所の指示に従い、全面的に調査に協力いたします」

今後の対応：「状況が分かり次第、随時ご報告させていただきます」

異物混入発生時のセリフ

【異物を発見した方への対応】

謝罪：「大変申し訳ございません。ご不快な思いをさせてしまいました」

確認：「お怪我やお体への影響はございませんか？」

現物確保：「恐れ入りますが、原因究明のため異物と料理を確認させてください」

対応：「すぐに代わりのお食事をご用意いたします」

【施設への報告】

報告：「本日○時頃、異物混入が発生いたしました」

原因説明：「原因を調査したところ、○○と判明いたしました」

再発防止：「再発防止策として○○を実施いたします」

お詫び：「ご心配をおかけし、申し訳ございませんでした」

急な欠勤・人員トラブル時のセリフ

【早朝に欠勤連絡を受けた時】

状況確認：「体調は大丈夫ですか？ゆっくり休んでください」

対応：「現場には私から連絡します。今日はお大事に」

現場への連絡：「○○さんが体調不良でお休みです。ヘルプを手配します」

ヘルプ依頼：「急で申し訳ないのですが、本日○○事業所のヘルプをお願いできますか」

【スタッフ間トラブルの相談を受けた時】

傾聴：「教えてくれてありがとうございます。詳しく聞かせてください」

確認：「それは辛かったですね。いつ頃から続いていますか？」

対応：「状況を確認して、対応を考えます。少しお時間ください」

フォロー：「また何かあったら、いつでも連絡してください」

チームで対応
