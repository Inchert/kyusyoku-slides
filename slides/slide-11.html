第11章 SV現場研修ガイド（SV候補者向け）

現場研修の意義

なぜ現場研修が必要か

  • 現場を知らなければ、現場を管理できない - 実際に調理・配膳を経験することで業務を理解
  • スタッフの気持ちがわかるようになる - 「大変さ」を体感することで信頼を得られる
  • 問題の本質が見えるようになる - 表面的な報告だけでなく、真の課題がわかる
  • 改善提案ができるようになる - 経験に基づいた具体的なアドバイスが可能に
  • 現場からの信頼を得られる - 「この人は現場をわかっている」と認められる

現場経験なしでSVになった場合の失敗例

  • 「なぜこんな簡単なことができないの？」 → 実際にやってみると簡単ではない
  • 「もっと効率よくやればいい」 → 現場の制約を知らないと机上の空論に
  • スタッフから信頼されない → 「あの人、現場のこと何も知らないくせに」
  • 本当の問題に気づかない → 表面的な対応しかできない
  • 施設への説明ができない → 「なぜそうなるのか」を説明できない

現場研修 6ヶ月間の目標

【知識】

  • 給食業務の全体像を理解する
  • 衛生管理の基本を完全に身につける
  • 各食種の特徴と調理法を理解する
  • 施設との連携方法を学ぶ
  • 帳票・記録の意味と書き方を覚える

【技術】

  • 基本的な調理作業ができるようになる
  • 盛り付け・配膳ができるようになる
  • 洗浄・清掃作業ができるようになる
  • 温度管理・記録ができるようになる

【マインド】

  • 現場スタッフの苦労を体感する
  • 食事を届ける喜びを実感する
  • チームワークの大切さを学ぶ
  • 「利用者のために」という意識を持つ

【人間関係】

  • 現場スタッフとの信頼関係を築く
  • 施設側スタッフとの関係を築く
  • 「一緒に働く仲間」として認められる

6ヶ月間のロードマップ

1ヶ月目：基本習得期

目標：「調理の基本ができる」

4ヶ月目：応用習得期

主菜の調理担当 / 複数作業を並行 / アレルギー対応実践 / 施設との連携

目標：「一通りの作業ができる」

5ヶ月目：リーダー体験期

シフトリーダー体験 / パートさんへの指示出し / トラブル対応経験 / 責任者の補佐

目標：「チームをまとめる経験をする」

業務別の学びポイント

下処理作業で学ぶこと

野菜：洗浄方法（3回洗い）、切り方（統一した大きさ・食べやすさ）、保管方法、廃棄量削減（歩留まり意識）

肉・魚：検収時チェック（品温・鮮度）、解凍方法（流水・冷蔵庫解凍）、二次汚染防止（まな板・包丁使い分け）、下味付け

学びのポイント：「なぜそうするのか」を考える、効率よい作業の工夫を観察、先輩の動きを真似る

調理作業で学ぶこと

大量調理の特徴：家庭料理との違い（量が多い、時間がかかる、段取り重要）、調理機器の使い方（スチコン・回転釜・フライヤー）、温度管理（中心温度75℃以上）、味付け調整（大量調理は味がぼやけやすい）

食種別調理：一般食と治療食の違い、嚥下調整食の作り方（ミキサー食・ソフト食・ゼリー食）、減塩・エネルギー調整のコツ、アレルギー対応（別鍋・別調理）

盛り付け・配膳で学ぶこと

盛り付け：分量の統一（計量意識）、見た目の美しさ（彩り・配置）、食種別盛付（嚥下食は特に見た目重要）、温度管理（温かいものは温かく）、トレイ配置ルール

配膳：配膳表確認（誰に何を届けるか）、名前・部屋番号確認（間違い防止）、アレルギー・禁止食品のダブルチェック、施設スタッフへの引き渡し、時間厳守

洗浄・清掃で学ぶこと

食器洗浄：下洗い→洗浄機→乾燥の流れ、洗浄温度確認（80℃以上のすすぎ）、残渣処理、洗浄機メンテナンス

厨房清掃：毎日清掃（床・作業台・機器）、週次清掃（壁・換気扇）、消毒方法（アルコール・次亜塩素酸）、グリストラップ清掃、害虫防止対策

食種の違いを理解する

↓

↓
全粥食 - 主食がお粥（ご飯が食べにくい）
↓

↓
嚥下調整食3 - やわらか食・まとまりやすい（飲み込み困難）
↓

↓
嚥下調整食1 - ゼリー・プリン状（飲み込みが非常に困難）
↓
嚥下調整食0 - 嚥下訓練用ゼリー（リハビリ段階）

患者・入所者の反応どの食種の人がどう食べているか観察

現場で信頼される新人になるために

基本姿勢

  • 謙虚に学ぶ姿勢
  • 「教えてください」と素直に言う
  • 言われたことは素直にやる
  • 同じミスを繰り返さない

コミュニケーション

  • 挨拶は自分から、大きな声で
  • 名前を覚えて呼ぶ
  • 「ありがとうございます」を忘れない
  • 雑談にも参加する（休憩時間等）

行動

  • 誰よりも早く動く
  • 汚れ仕事、きつい仕事を進んでやる
  • 時間を守る（遅刻厳禁）
  • メモを取る

注意すべきこと

  • 「前の職場では〜」と言わない
  • 「知ってます」と言わない
  • 先輩の仕事を批判しない
  • 最初から改善提案しない

ベテランパートさんとの付き合い方

パートさんは「現場のプロ」

  • 何年も同じ現場で働いている
  • その施設のことを一番よく知っている
  • 調理の腕は一流の人も多い

だから：敬意を持って接する / 「教えてください」とお願いする / 上から目線は絶対NG / 「将来SVになる」は言わない方が無難 / 味方になってもらえると心強い

現場研修での正しい立ち位置

結論：「まだSVじゃない顔」で入る

指示しない / 正解を押し付けない / 仕切らない

まずは "現場の一員（ただし観察力高め）" がベスト

正しい最初の挨拶

「SVとして来ましたが、まずは現場をしっかり理解したくて来ています」

→ この一言で警戒心が9割消えます

NGな挨拶

  • 「改善点を見に来ました」
  • 「やり方を見直します」
  • 「本部の指示で」

給食業界ならではの"地雷"を踏まない

給食現場の特徴：ベテラン女性が多い / 独自ルール・暗黙知が強い / 安全・衛生が最優先 / 忙しい時間帯は殺気立つ

立ち振る舞いの基本：

  • 作業を止めて質問しない
  • ピークタイムは話しかけない
  • 「それ、危なくないですか？」は言わない
  • 衛生ルールの運用実態を観察
  • 曜日・献立による差を把握

→ マニュアルより現実を見る

研修中に観察すべき5つの視点

【人】

  • 誰がキーパーソンか（責任者以外にも影響力ある人）
  • スタッフ同士の関係性（派閥、仲良しグループ）
  • 困ったとき誰に聞いているか
  • 新人への態度はどうか

【モノ】

  • 設備の使い方（マニュアル通りか独自か）
  • 道具の配置・動線
  • 食材の保管状態
  • 掲示物（古いまま放置されてないか）

【時間】

  • どの時間帯が最も忙しいか
  • 休憩はいつ、どのように取っているか
  • 納品・検品のタイミング
  • 余裕がある時間帯はいつか

【ルール】

  • マニュアルと実態のギャップ
  • 暗黙の了解（「これはこうやるもの」）
  • 衛生管理の徹底度
  • 記録・帳票の運用状況

【雰囲気】

  • 現場の空気感（ピリピリ？和やか？）
  • コミュニケーションの取り方
  • 問題が起きたときの対処の仕方
  • 施設側との関係性

→ 観察したことはその日のうちにメモ

質問の仕方

NGな聞き方

NG例なぜダメか
「なんでこのやり方なんですか？」否定に聞こえる
「マニュアルと違いますよね？」指摘・批判に聞こえる
「これ、効率悪くないですか？」上から目線
「前の現場ではこうでしたけど」比較は嫌われる

OKな聞き方

OK例効果
「これ、どうやるんですか？」素直に教わる姿勢
「コツとかあったら教えてください」相手を立てる
「〇〇さんすごいですね」承認＋興味
「ここの現場ならではの工夫は？」現場へのリスペクト

質問のタイミング：○作業が落ち着いた後 / ○休憩中（相手の様子を見て） / ○片付けしながら / ×調理中・配膳中 / ×忙しそうなとき

信頼を得る3原則

原則1: まず動く

  • 「何かやれることありますか？」
  • 雑用でも喜んでやる
  • 洗い物・運搬・清掃を進んでやる

見ているだけ→「何しに来たの？」

原則2: 覚える努力を見せる

  • メモを取る（見えるところで）
  • 名前を覚える（できれば初日で全員）
  • 教わったことは2回目から自分でやる

原則3: 感謝を伝える

  • 「ありがとうございます」を惜しまない
  • 「〇〇さんのおかげで分かりました」
  • 教えてもらったことを実践→報告

困った場面と対処法

明らかに間違ったやり方をしている

NG その場で指摘する / 「それ違いますよ」

OK 黙って観察してメモ / 後日責任者に「確認」として聞く / 安全に関わる場合のみ責任者にそっと相談

ベテランに冷たくされる

NG 避ける・距離を置く / 上司に報告して何とかしてもらおうとする

OK 淡々と丁寧に接し続ける / その人の仕事を認める発言 / 時間をかけて関係構築

何をしていいか分からない

NG ぼーっと立っている / スマホを見る

OK 「次、何かやれることありますか？」 / 周りを見てできそうな片付け・清掃

ミスをしてしまった

NG 隠す・ごまかす / 言い訳をする

OK すぐに報告 / 「すみません、〇〇してしまいました」 / リカバリーを一緒にやる / 同じミスは絶対にしない

衛生上の問題を発見した

NG 全員の前で指摘する / いきなり本部に報告

OK 責任者に個別に相談 / 「ちょっと確認したいことが…」 / 重大な問題は上長に報告（現場には配慮）

→ 「正しいこと」より「正しい伝え方」

研修終了時に言うと強い一言

「今日までありがとうございました。
ここで教えてもらったやり方を
どう活かせるか、ちゃんと考えます」

→ これで 「敵じゃないSV」認定 されます

さらに効果的な行動：全員に個別で挨拶・お礼 / 「〇〇さんに教えてもらった△△、勉強になりました」と具体的に / 後日お礼のメッセージを責任者経由で伝える

まとめ：現場研修で得られるもの

給食SVは現場に嫌われたら詰み、信頼されたら無双

最初の研修は「評価されに行く場」じゃなく「信頼を作る場」

得るべきでないもの

  • 改善点のリスト
  • 問題点の報告書
  • 「自分はできる」という自信

得るべきもの

  • 「あの人なら話を聞いてくれそう」という信頼
  • 現場のリアルな動きの理解
  • キーパーソンとの関係性
  • 「また来てほしい」と思われる存在感

ポイント：ぼーっと立たない、自分から動く

「コツとかあったら教えてください」

【教えてもらった後】

【最終日の挨拶】

「今日までありがとうございました。ここで教えてもらったやり方を、どう活かせるか、ちゃんと考えます」

「〇〇さんに教えていただいた△△、すごく勉強になりました。ありがとうございました」

よくある困った場面と切り返し

困った場面 悪い対応例 良い対応例
「SVになるんでしょ？」と聞かれた 「はい、そうです」「いずれ皆さんを管理する立場に」 「まずは現場をしっかり理解したくて来ています。教えてください」
「これ、違うやり方だけど」と言われた 「マニュアルではこうですよね？」「前の現場ではこうでした」 「なるほど、ここのやり方を教えてください」「何か理由があるんですか？勉強になります」
ベテランに冷たくされた 避ける、距離を置く、上司に相談 淡々と丁寧に接し続ける「〇〇さんの段取りすごいですね」と仕事を認める
衛生上の問題を見つけた 「それ危なくないですか？」と指摘、本部に報告 メモして後で責任者に「確認したいことがあるのですが…」と相談
忙しそうで質問できない 忙しい時に話しかける、ピーク時に質問 片付け中や休憩時間に「さっきの件、ちょっと教えていただけますか？」
「若いのに大変ね」と言われた 「いえ、全然」「慣れてます」 「皆さんの方がずっと大変ですよね。尊敬します」

絶対にやってはいけないこと（現場での地雷）

言ってはいけない一言

  • 「前の現場では〜」（比較発言）
  • 「それ、効率悪くないですか？」
  • 「マニュアルと違いますよね？」
  • 「なんでこのやり方なんですか？」
  • 「改善した方がいいと思います」
  • 「本部に報告しておきます」

やってはいけない行動

  • ピーク時に話しかける
  • 作業を止めて質問する
  • スマホをいじる（休憩中でも見られている）
  • 疲れた顔・退屈そうな顔をする
  • 現場の悪口を他で言う
  • ぼーっと立っている
